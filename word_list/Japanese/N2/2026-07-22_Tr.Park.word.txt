設備
설비(せつび)
設 (베풀 설) = 言 (말씀 언) + 殳 (몽둥이 수) : 말을 꺼내거나 도구를 갖추어 베풀고 설치한다는 뜻입니다.
備 (갖출 비) = 亻 (사람 인) + 𠂇 (손) + 田 (밭 전) + 夂 (천천히 걸을 쇠) : 필요한 도구나 사람을 에워싸서 완벽하게 갖추어 둔다는 뜻입니다.
어떤 목적을 위해 필요한 도구나 시설을 알맞게 베풀어 갖추는 것을 의미합니다.

備える
갖추다, 대비하다(そなえる)
備 (갖출 비) = 亻 (사람 인) + 𠂇 (손) + 田 (밭 전) + 夂 (천천히 걸을 쇠) : 필요한 것들을 미리 완벽하게 준비해 둔다는 뜻입니다.
앞으로 일어날 일이나 필요한 상황에 대비하여 물건이나 마음의 준비를 해두는 것을 뜻하는 동사입니다.

節約
절약(せつやく)
節 (마디 절) = ⺮ (대죽) + 即 (곧 즉) : 대나무(⺮)의 마디처럼 지나치지 않고 알맞게 끊어 조절한다는 뜻입니다.
約 (맺을 약) = 纟 (실 사) + 勺 (구기 작) : 실로 물건을 팽팽하게 묶어 낭비하지 않도록 조이거나 줄인다는 뜻입니다.
지나침이 없도록 마디를 짓듯(節) 팽팽하게 조여서(約) 낭비를 줄이고 아끼는 것을 의미합니다.

季節
계절(きせつ)
季 (끝 계) = 禾 (벼 화) + 子 (아들 자) : 벼(禾)의 수확 시기나 한 해의 끝과 시작을 이루는 철을 뜻합니다.
節 (마디 절) = 대나무의 마디처럼 시간이나 자연의 흐름을 일정하게 나누는 구분을 뜻합니다.
일 년의 시간 흐름 속에서 자연 현상이나 기후에 따라 마디를 나누어(節) 구분한 특정한 철(季)을 의미합니다.

相互
상호(そうご)
相 (서로 상) = 木 (나무 목) + 目 (눈 목) : 나무의 상태를 눈으로 자세히 살펴본다는 뜻에서 서로를 뜻합니다.
互 (서로 호) = 끈이나 실을 얼기설기 교차시켜 감아놓은 모습에서 서로 얽혀있음을 의미합니다.
두 개 이상의 대상이 서로(相) 얽혀서(互) 주고받는 관계를 의미합니다.

相談
상담(そうだん)
相 (서로 상) = 나무를 눈으로 살펴보듯 서로를 마주한다는 뜻입니다.
談 (말씀 담) = 言 (말씀 언) + 炎 (불 화) : 불(炎)이 타오르듯 열정적으로 말(言)을 주고받으며 의논한다는 뜻입니다.
어떤 문제를 해결하기 위해 서로(相) 마주 앉아 의견을 나누고 의논하는(談) 것을 의미합니다.

相手
상대, 상대방(あいて)
相 (서로 상) = 마주 보는 대상이나 관계를 뜻합니다.
手 (손 수) = 사람의 손 모양을 본뜬 상형자로 도움이나 상대를 뜻합니다.
나와 마주 보고 있는 상대방(相)이나 손(手)을 마주 잡고 협력 또는 대립하는 관계의 사람을 의미합니다.

お互い
상호간, 서로(おたがい)
互 (서로 호) = 끈이 서로 엇갈리듯 대상을 교차하여 가리키는 글자입니다.
누구 한쪽이 아니라 우리들 모두 혹은 양쪽이 서로 똑같이 해당되는 상태를 뜻하는 단어입니다.

存在
존재(そんざい)
存 (있을 존) = 子 (아들 자) + 才 (바탕 재) : 사람이 세상에 태어나 바탕을 가지고 살아 있다는 뜻입니다.
在 (있을 재) = 土 (흙 토) + 才 (바탕 재) : 흙 위에 실체를 가지고 자리 잡고 있다는 뜻입니다.
사물이나 사람이 세상에 생명이나 실체를 가지고(存) 자리 잡아 있는(在) 상태를 뜻합니다.

探索
탐색(たんさく)
探 (찾을 탐) = 扌 (손 수) + 罙 (깊을 심) : 손(扌)을 아주 깊은(罙) 곳까지 뻗어 더듬어 찾는다는 뜻입니다.
索 (찾을 색) = 십(十)과 덮을 멱(冖), 실 사(糸)가 합쳐져 뭉친 실을 풀어 헤친다는 뜻입니다.
숨겨진 사물이나 알 수 없는 내용을 깊이 더듬고(探) 뒤져서 찾아내는(索) 것을 의미합니다.

探す
찾다(さがす)
探 (찾을 탐) = 扌 (손 수) + 罙 (깊을 심) : 손을 뻗어 깊은 곳까지 더듬어 찾는다는 뜻입니다.
어디에 있는지 모르는 물건이나 사람을 눈으로 보거나 손으로 더듬어 발견하려고 애쓰는 것을 뜻하는 동사입니다.

担当
담당(たんとう)
担 (멜 담) = 扌 (손 수) + 旦 (아침 단) : 손으로 무거운 짐을 어깨에 둘러메거나 떠맡는다는 뜻입니다.
当 (마땅할 당) = 햇볕이 공평하게 비추듯 맡은 바 직분에 딱 들어맞는다는 뜻입니다.
자신이 맡은 바 책임을 짊어지고(担) 그 직무나 일을 마땅히 도맡아(当) 처리하는 것을 의미합니다.

担任
담임(たんにん)
担 (멜 담) = 손으로 무거운 책임을 짊어지거나 떠맡는다는 뜻입니다.
任 (맡길 임) = 亻 (사람 인) + 壬 (짊어질 임) : 사람에게 짐을 짊어지게 하여 책임을 맡긴다는 뜻입니다.
특정한 학급이나 업무에 대한 책임을 짊어지고(担) 오롯이 맡아(任) 관리하는 사람을 뜻합니다.

担う
짊어지다, 떠맡다(になう)
担 (멜 담) = 扌 (손 수) + 旦 (아침 단) : 어깨나 손에 무거운 짐을 짊어지는 모습입니다.
물건을 어깨에 메거나, 조직이나 사회에서 중요한 책임과 임무를 몸소 떠맡는 것을 뜻하는 동사입니다.

知識
지식(ちしき)
知 (알 지) = 矢 (화살 시) + 口 (입 구) : 화살처럼 빠르게 입으로 말할 수 있을 만큼 사물을 명확히 아는 뜻입니다.
識 (알 식) = 言 (말씀 언) + 戠 (기록할 지) : 기호나 말로 기록하여 사물을 분별하고 안다는 뜻입니다.
사물의 이치를 깨달아 알고(知) 분별하여 내 것으로 만드는(識) 것을 의미합니다.

知恵
지혜(ちえ)
知 (알 지) = 사물을 명확히 꿰뚫어 아는 능력을 뜻합니다.
恵 (은혜 혜) = 心 (마음 심) + 匸 (감출 은) + 示 (제단) : 마음속에서 우러나오는 너그러운 은혜나 슬기를 뜻합니다.
사물의 이치를 빠르게 깨닫는 지식(知)과 그것을 바르게 활용하여 문제를 해결하는 슬기로운 마음(恵)을 뜻합니다.

恵まれる
은혜를 입다, 풍족하다(めぐまれる)
恵 (은혜 혜) = 마음속에서 베풀어지는 너그러운 은혜나 사랑을 뜻합니다.
좋은 환경이나 조건, 재능 등을 하늘이나 타인으로부터 베풀어져(恵) 풍족하게 얻은 상태를 뜻하는 동사입니다.

認識
인식(にんしき)
認 (알 인) = 言 (말씀 언) + 忍 (참을 인) : 참고 꼼꼼히 살핀 뒤 말로 인정한다는 뜻입니다.
識 (알 식) = 사물을 분별하고 뚜렷하게 아는 것을 뜻합니다.
어떤 사실이나 상황을 분명히 알고(認) 마음에 새겨서 확실하게 분별하는(識) 것을 의미합니다.

確認
확인하다(かくにん)
確 (굳을 확) = 石 (돌 석) + 隺 (두루미 학) : 두루미가 돌처럼 단단하게 자리 잡고 있는 모습에서 확실하다는 뜻입니다.
認 (알 인) = 꼼꼼히 살핀 뒤 말로 인정하고 확인한다는 뜻입니다.
어떤 사실이나 상태가 틀림이 없는지 굳게(確) 짚어보고 맞다고 인정하는(認) 것을 의미하는 동사입니다.

認める
인정하다, 받아들이다(みとめる)
認 (알 인) = 꼼꼼히 살핀 뒤 말로 확실하게 확인하고 받아들인다는 뜻입니다.
눈으로 보고 그 존재나 사실을 확실하게 마음으로 받아들여 인정하거나 허락하는 것을 뜻하는 동사입니다.

燃焼
연소(ねんしょう)
燃 (탈 연) = 火 (불 화) + 然 (그러할 연) : 불(火)이 기세 좋게 타오른다는 뜻입니다.
焼 (사를 소) = 火 (불 화) + 尭 (높을 요) : 불이 가마를 달구어 뜨겁게 태우고 사른다는 뜻입니다.
물질이 빛이나 열을 내며 불길에 타오르는(燃) 현상을 의미합니다.

燃料
연료(ねんりょう)
燃 (탈 연) = 불이 타오르는 성질을 뜻합니다.
料 (헤아릴 료) = 米 (쌀 미) + 斗 (말 두) : 곡식을 셈하여 덜어내듯 재료나 분량을 뜻합니다.
불에 태워 열이나 동력을 얻기 위해 사용하는 재료(料)를 의미합니다.

燃える
타다(もえる)
燃 (탈 연) = 화염(火)이 타오르는 상태를 뜻합니다.
물질에 불이 붙어 활활 타오르거나, 마음속에 열정이 뜨겁게 솟구치는 상태를 뜻하는 동사입니다.

拝借
배차, 빌림(はいしゃく)
拝 (절 배) = 扌 (손 수) + 丰 (머리 숙인 모습) : 손을 모으고 머리를 숙여 공경하는 뜻입니다.
借 (빌릴 차) = 亻 (사람 인) + 昔 (옛 석) : 남의 물건을 옛날부터 빌려 쓰고 돌려준다는 뜻입니다.
상대방의 물건 등을 공경하는 마음으로 공손하게 빌려(借) 쓰는(拝) 것을 의미하는 비즈니스 용어입니다.

拝む
우러러보다, 예배하다(おがむ)
拝 (절 배) = 손을 모으고 머리를 숙여 경외심을 표하는 모습입니다.
신이나 부처 등 존귀한 대상 앞에서 두 손을 모으고 머리를 숙여 간절히 빌거나 우러러보는 것을 뜻하는 동사입니다.

借
빌릴 차(차/카리루)
借 (빌릴 차) = 亻 (사람 인) + 昔 (옛 석) : 다른 사람의 물건이나 돈을 빌려 쓰고 나중에 되돌려준다는 뜻입니다.
남의 물건이나 돈을 대가를 치르거나 약속하고 잠시 빌려오는 것을 뜻하는 단일 한자어입니다.

祈る
기도하다(いのる)
祈 (빌 기) = 礻 (보일 시) + 斤 (도끼 근) : 제단(礻)에 제물을 올리고 간절히 바라는 바를 빈다는 뜻입니다.
신이나 절대자에게 바라는 일이 이루어지기를 마음 다해 간절히 빌고 기원하는 동사입니다.

願う
원하다, 부탁하다(ねがう)
願 (원할 원) = 原 (언덕 원) + 頁 (머리 혈) : 머리를 숙여 묵직하게 마음속 소원을 빈다는 뜻입니다.
자신의 마음속에 품은 소망이 이루어지기를 바라거나 남에게 정중하게 부탁하는 것을 뜻하는 동사입니다.

望む
바라다, 희망하다(のぞむ)
望 (바랄 망) = 亡 (망할 망/멀리) + 月 (달 월) + 壬 (천간 임) : 멀리 있는 달을 바라보며 기대한다는 뜻입니다.
먼 곳을 바라보듯 앞으로의 일이나 미래에 대해 특정한 것을 간절히 바라거나 기대하는 동사입니다.

破壊
파괴(はかい)
破 (깨뜨릴 파) = 石 (돌 석) + 皮 (가죽 피) : 돌로 가죽을 쳐서 산산조각 낸다는 뜻입니다.
壊 (무너질 괴) = 土 (흙 토) + 褱 (품을 회) : 형태를 갖춘 것이 허물어진다는 뜻입니다.
형체를 갖춘 사물이나 체제를 깨뜨리고(破) 허물어뜨려(壊) 못 쓰게 만드는 것을 뜻합니다.

破産
파산(はさん)
破 (깨뜨릴 파) = 깨뜨려 무너뜨린다는 뜻입니다.
産 (낳을 산/재산 산) = 生 (날 생) + 竔 (설 립) : 재산이나 생업을 뜻합니다.
가지고 있던 재산(産)이나 경제적 기반이 완전히 깨지고 무너져(破) 더 이상 사업을 지속할 수 없는 상태를 뜻합니다.

破れる
찢어지다, 패하다(やぶれる)
破 (깨뜨릴 파) = 돌로 쳐서 가죽을 찢고 부순다는 뜻입니다.
천이나 종이 등이 힘에 의해 찢어져 구멍이 나거나, 시합이나 싸움에서 상대에게 패하여 무너지는 것을 뜻하는 동사입니다.

迷惑
민폐, 신뢰를 해침(めいわく)
迷 (미혹할 미) = 辶 (책받침) + 米 (쌀 미) : 길이 헷갈려 갈피를 잡지 못하고 헤맨다는 뜻입니다.
惑 (미혹할 혹) = 或 (혹시 혹) + 心 (마음 심) : 마음이 헷갈리고 현혹된다는 뜻입니다.
길을 잃고 헤매듯(迷) 마음이 헷갈리고 괴로워지게(惑) 하여 남에게 피해를 끼치는 것을 뜻합니다.

迷惑をかける
민폐를 끼치다(めいわくをかける)
迷惑 (미혹) = 남을 곤란하게 함.
掛ける (걸다) = 상대방에게 부담이나 짐을 얹음.
상대방에게 곤란한 상황이나 부담을 주어 정신적, 물리적인 피해를 끼치는 행동을 뜻하는 표현입니다.

迷子
미아(まいご)
迷 (미혹할 미) = 길을 잃고 헤매다는 뜻입니다.
子 (아들 자) = 어린아이를 뜻하는 단어입니다.
넓은 장소에서 길을 잃고(迷) 헤매어 보호자를 찾지 못하는 아이(子)를 뜻합니다.

迷う
길을 헤매다, 망설이다(まよう)
迷 (미혹할 미) = 사방이 헷갈려 방향을 잡지 못하고 헤매는 모습입니다.
길을 잃고 헤매거나, 여러 선택지 앞에서 마음의 결정을 내리지 못하고 우물쭈물 망설이는 것을 뜻하는 동사입니다.

免許
면허(めんきょ)
免 (면할 면) = 투구를 쓴 사람의 모습에서 위험이나 금지된 상태를 벗어난다는 뜻입니다.
許 (허락할 허) = 言 (말씀 언) + 午 (낮 오) : 밝은 곳에서 말로써 확실하게 승낙하고 허락한다는 뜻입니다.
금지되었던 특정한 행위를 면제하여(免) 자유롭게 할 수 있도록 공식적으로 허락하는(許) 것을 의미합니다.

어떤 행위를 하도록 마땅하다고 여겨(可) 공식적으로 허락하고 승낙하는(許) 것을 의미합니다.

優しい

능력이나 품질 등이 다른 것보다 훨씬 뛰어나고 훌륭한 상태를 뜻하는 동사입니다.

秀 (빼어날 수) = 𥝌 (벼 화의 변형) + 乃 (이에 내) : 벼 이삭이 다른 것들보다 훨씬 빼어나게 잘 자란 모습입니다.
재능이나 성적, 품질 등이 남달리 뛰어나고(優) 매우 빼어난(秀) 상태를 뜻합니다.

岩 (바위 암) = 山 (뫼 산) + 石 (돌 석) : 산에 있는 단단한 바위를 뜻합니다.

中 (가운데 중) = 한가운데를 정확히 꿰뚫는 뜻입니다.
어떤 목적지로 가는 길(途)의 한가운데(中), 즉 일이나 이동을 하던 도중에 있는 상태를 의미합니다.

解 (풀 해) = 角 (뿔 각) + 刀 (칼 도) + 牛 (소 우) : 소의 뿔을 칼로 쪼개어 얽힌 것을 푼다는 뜻입니다.
사물의 이치(理)를 논리적으로 쪼개고 풀어내어(解) 온전히 깨닫는 것을 의미합니다.

理由

由 (말미암을 유) = 식물의 싹이 아래에서 위로 트여 나오는 모양에서 유래하여 무엇으로부터 비롯됨을 뜻합니다.
어떤 결과나 행동이 생겨나게 된 정당한 이치(理)와 그로 말미암아 된 까닭(由)을 의미합니다.

정해진 상시적인 규칙이 아니라, 그때그때의 사정에 임하여(臨) 일시적으로 정하는 때(時)를 의미합니다.

流布

정보나 소문, 물건 등이 물결치듯(流) 세상에 널리 퍼져서 사람들에게 알려짐(布)을 의미합니다.

옷의 여밈이나 물건을 가볍게 묶을 때 사용하는 얇은 끈이나 노끈을 뜻하는 단일 한자어입니다.

특정한 사람의 태도나 행동이 마음에 들지 않아 강하게 싫어하고 미워하는 감정을 뜻하는 동사입니다.

憎い

표적이나 목표물을 명중시키거나 차지하기 위해 눈을 떼지 않고 기회를 노리는 행위를 뜻하는 동사입니다.

結論

受ける

타인이 주는 물건, 사랑, 시험, 피해 등을 몸이나 마음으로 고스란히 받아들이는 것을 뜻하는 동사입니다.

乗り越す
지나치다, 극복하다(のりこす)
乗る (타다) = 탈것에 오름.
越す (넘다) = 공간이나 시기의 경계를 넘어섬.
전철 등을 타고 가다가 내려야 할 정거장을 지나쳐 더 멀리 가거나, 난관을 타고 넘는 것을 뜻하는 복합 동사입니다.

越す
넘다, 지나다(こす)
越 (넘을 월) = 走 (달릴 주) + 戉 (도끼 월) : 경계나 산을 뛰어넘어 다른 곳으로 건너간다는 뜻입니다.
물리적인 산이나 고개를 넘어가거나, 특정한 시기나 세월을 무사히 지나 보내는 것을 뜻하는 동사입니다.

張り切る
의욕이 넘치다, 긴장하다(はりきる)
張る (팽팽하게 당기다) + 切る (끝까지 자르다)
활시위를 팽팽하게 당긴 것처럼 긴장감이 극에 달해 의욕과 활력이 가득 차오르는 상태를 뜻하는 복합 동사입니다.

目立つ
눈에 띠다(めだつ)
目 (눈 목) = 사람의 눈.
立つ (서다) = 땅 위에 바로 서다.
다른 것들 사이에서 눈에 띄게 드러나서 남들의 시선을 쉽게 끄는 상태를 뜻하는 동사입니다.

持ち越す
다음으로 넘기다, 미루다(もちこす)
持つ (가지다) + 越す (넘어가다)
현재 해결하지 못한 일이나 남은 문제를 손에 쥔 채로 다음 기회나 시기로 미루어 넘기는 복합 동사입니다.

見送る
배웅하다, 보류하다(みおくる)
見る (보다) + 送る (보내다)
떠나는 사람을 눈으로 배웅하며 보거나, 당장의 결정을 미루고 상황을 지켜보며 보류하는 복합 동사입니다.

見合わせる
마주보다, 보류하다(みあわせる)
見る (보다) + 合わせる (합치다)
서로 얼굴을 마주 보고 눈빛을 교환하거나, 실행하려던 계획을 서로의 의견을 확인하며 잠시 보류하는 복합 동사입니다.

悔しい
분하다, 억울하다(くやしい)
悔 (뉘우칠 회) = 忄 (마음 심) + 毎 (매양 매) : 마음속에 늘 후회와 미련이 남는다는 뜻입니다.
일이 뜻대로 되지 않아 억울하고 분한 마음이 가시지 않는 상태를 뜻하는 형용사입니다.

ずうずうしい
뻔뻔스럽다(ずうずうしい)
図々しい의 히라가나 표기로, 염치없고 뻔뻔한 태도를 뜻합니다.
자신이 나서야 할 곳과 아니어야 할 곳을 분간하지 못하고 철면피처럼 행동하는 성질을 뜻하는 형용사입니다.

厚かましい
뻔뻔하다(あつかましい)
厚 (두터울 후) = 厂 (기슭 엄) + 曰 (가로 왈) + 子 (아들 자) : 두껍고 넉넉하다는 뜻입니다.
얼굴 가죽이 두껍다(厚)는 비유에서 유래하여, 염치없이 뻔뻔하고 당돌한 성질을 뜻하는 형용사입니다.

ずるい
교활하다(ずるい)
狡い의 히라가나 표기로, 잔머리를 굴리는 성질을 뜻합니다.
자신의 이익만을 위해 꾀를 부리며 남을 속이거나 정당하지 않게 행동하는 교활한 태도를 뜻하는 형용사입니다.

鋭い
날카롭다(するどい)
鋭 (날카로울 예) = 金 (쇠 금) + 兌 (기뻐할 탈) : 쇠붙이의 날이 시퍼렇게 서서 날카롭다는 뜻입니다.
칼날 등이 매우 날카롭거나, 사람의 통찰력이나 감각이 빈틈없이 예리하고 예민한 상태를 뜻하는 형용사입니다.

頼もしい
믿음직하다(たのもしい)
頼 (의지할 뢰) = 束 (묶을 속) + 刀 (칼 도) + 貝 (조개 패) : 서로 돕고 기대게 한다는 뜻입니다.
마음 놓고 의지(頼)할 수 있을 만큼 능력이 뛰어나고 든든하게 느껴지는 성질을 뜻하는 형용사입니다.

頼む
부탁하다, 의뢰하다(たのむ)
頼 (의지할 뢰) = 남에게 마음을 기대고 의지한다는 뜻입니다.
상대방에게 도움이나 일을 해달라고 간절히 부탁하거나 의뢰하는 것을 뜻하는 동사입니다.

大げさ
과장됨, 야단스러움(おおげさ)
大 (클 대) + 거창한 모양을 나타내는 일본어 표현이 결합된 형태입니다.
실제보다 훨씬 부풀려서 대단하고 유난스럽게 행동하거나 표현하는 상태를 뜻하는 나형용사입니다.

快適
쾌적함(かいてき)
快 (쾌할 쾌) = 忄 (마음 심) + 夬 (터놓을 쾌) : 막힌 곳이 뚫려 마음이 시원하다는 뜻입니다.
適 (맞을 적) = 상황이나 조건에 꼭 들어맞는다는 뜻입니다.
주변 환경이나 기분이 시원하고(快) 알맞게 꼭 들어맞아(適) 상쾌하고 편안한 상태를 뜻하는 나형용사입니다.

雇用
고용(こよう)
雇 (품팔이 고) = 戸 (지게문 호) + 隹 (새 추) : 사람을 고용하여 대가를 치르고 일을 시킨다는 뜻입니다.
用 (쓸 용) = 사람이나 도구를 유용하게 사용한다는 뜻입니다.
노동력을 필요로 하는 곳에서 대가를 지불하기로 하고 사람을 써서 일하게 하는 것을 의미합니다.

くたくた
녹초가 됨, 몹시 지침(くたくた)
몸이 피로에 지쳐 기운이 하나도 없고 늘어지거나 물건이 낡아서 흐물흐물해진 모양을 나타내는 의태어입니다.

ぐたぐた
구차하게 늘어놓음, 툴툴거림(ぐたぐた)
뚜렷한 결론 없이 불평이나 쓸데없는 말을 늘어지게 자꾸 반복하는 모양을 나타내는 의태어입니다.

稀
드묾, 희귀함(まれ)
稀 (드물 희) = 禾 (벼 화) + 希 (바랄 희) : 존재하거나 일어나는 빈도가 매우 적고 드물다는 뜻입니다.
보통의 경우와 달리 찾아보기 힘들 정도로 드물고 희귀한 상태를 뜻하는 나형용사입니다.

ずきずき
지끈지끈, 쑤시듯 아픈 모양(ずきずき)
상처나 두통 등으로 인해 맥박이 뛰는 것에 맞춰 통증이 주기적으로 울리며 쑤시는 아픔을 나타내는 의성·의태어입니다.

すっかり
완전히, 남김없이(すっかり)
밑바닥까지 다 드러난 상태를 뜻하는 부사입니다.
어떤 변화나 잊어버림이 조금도 남김없이 전부 완벽하게 이루어진 상태를 뜻합니다.

ずらい
하기 어렵다, 곤란하다(~づらい) - ~づらい의 오기
동사어간에 붙어 어떤 행동을 하기가 심리적·물리적으로 불편하고 곤란함을 나타내는 접미사입니다.

ぴたり
딱 맞는 모양, 뚝 그치는 모양(ぴたり)
흔들림 없이 기준에 정확히 들어맞거나, 움직임이 단번에 멈추는 모양을 나타내는 의태어입니다.

ぴったり
딱 맞음, 아주 잘 어울림(ぴったり)
두 사물이 틈 없이 밀착되거나 크기나 조건 등이 한 치의 어긋남 없이 완벽하게 들어맞는 모양을 뜻하는 부사·나형용사입니다.

ことわない
거절하다(ことわらない)의 어근 또는 오기 -> 断る(ことわる)
断 (끊을 단) = 칼로 단호하게 잘라낸다는 뜻입니다.
상대방의 부탁이나 제안을 정중하게 거절하거나 사양하는 것을 뜻하는 동사입니다.

配る
나눠주다, 배포하다(くばる)
配 (짝 배) = 酉 (닭 유) + 己 (몸 기) : 술을 공평하게 나누어 따른다는 뜻에서 유래했습니다.
물건이나 종이 등을 여러 사람에게 골고루 나누어주거나 배포하는 행위를 뜻하는 동사입니다.

心配
걱정, 염려(しんぱい)
心 (마음 심) = 마음의 활동을 뜻합니다.
配 (나눌 배/생각할 배) = 마음을 이리저리 나누어 쓰며 염려한다는 뜻입니다.
마음(心)이 편치 못하고 이리저리 쓰이며(配) 안 좋은 일이 생길까 봐 염려하고 걱정하는 것을 의미합니다.

詫びる
사과하다(わびる)
詫 (속일 타/꾸짖을 타) = 言 (말씀 언) + 宅 (집 댁) : 자신의 잘못을 인정하고 용서를 빈다는 뜻으로 쓰임.
자신의 잘못이나 실수에 대해 상대방에게 고개를 숙여 미안하다고 사과하는 것을 뜻하는 동사입니다.

〜せざるを得ない
~하지 않을 수 없다(~せざるを得ない)
문법 표현으로, 의지와 상관없이 어쩔 수 없이 어떤 행동을 해야만 하는 상황을 나타내는 관용 구문입니다.

ざるを得ない
~하지 않을 수 없다(ざるを得ない)
동사의 부정형(~ない)에서 'ない' 대신 'を得ない'가 붙어, 피할 수 없는 강제성을 나타내는 문법 표현입니다.

諦める
포기하다, 단념하다(あきらめる)
諦 (살필 체) = 言 (말씀 언) + 帝 (임금 제) : 이치를 훤히 밝혀 명백히 살핀다는 뜻에서 유래했습니다.
어떤 일에 대해 더 이상의 미련이나 가망이 없다고 여겨 마음을 깨끗이 접고 포기하는 것을 뜻하는 동사입니다.

しかない
~할 수밖에 없다(~しかない)
어떤 선택지나 방법이 그것 하나밖에 남아 있지 않아 오직 그 길로만 갈 수 있음을 나타내는 문법 표현입니다.

〜からしわ
문맥상 오류 또는 ~から(～때문에/로부터)의 변형 표현입니다.

直す
고치다(なおす)
直 (곧을 직) = 곧고 바른 상태를 뜻합니다.
어긋나거나 잘못된 것을 원래의 바른 상태로 되돌려 수리하거나 고치는 것을 뜻하는 동사입니다.

〜次第
~하는 대로, ~하기 나름이다(~しだい)
次第 (차제 차) = 순서나 계기를 뜻합니다.
명사 뒤에 붙어 그 일이 끝나자마자 곧바로 다음 행동을 함을 나타내거나, 앞의 내용에 따라 결과가 좌우됨을 나타내는 문법 표현입니다.

製品
제품(せいひん)
製 (지을 제) = 制 (절제할 제) + 衣 (옷 의) : 재료를 다듬어 물건을 대량으로 제조한다는 뜻입니다.
品 (물건 품) = 口 (입 구) 세 개가 모여 수많은 물건을 뜻합니다.
원재료를 공장에서 가공하여 사람들에게 판매하기 위해 만들어 낸 완성된 물건을 의미합니다.

品物
물건, 상품(しなもの)
品 (물건 품) = 여러 가지 종류의 사물이나 물건을 뜻합니다.
物 (만물 물) = 세상에 존재하는 구체적인 형체의 사물을 뜻합니다.
사람들이 일상에서 사용하거나 거래하는 구체적인 형태를 가진 물건이나 상품을 의미합니다.

商品
상품(しょうひん)
商 (장사 상) = 亠 (돼지머리 해) + ⱚ + 口 : 시장에서 이익을 남기기 위해 거래하는 상업적 행위와 연관됨.
品 (물건 품) = 거래의 대상이 되는 물건을 뜻합니다.
시장에서 매매하기 위해 만들어진 물건이나 재화를 의미합니다.

n + 次階 / 次第
~하기 나름이다(～しだい)
앞에 오는 명사나 조건의 상태에 따라 결과가 전적으로 좌우되고 결정됨을 나타내는 표현입니다.

末に
~끝에(すとに) -> 末に(すとに / まさに)
末 (끝 말) = 나무의 윗부분이나 어떤 기간의 맨 마지막을 뜻합니다.
오랜 시간 동안 고민하거나 노력한 맨 마지막 시점이나 그 결과의 끝을 나타내는 문법 표현입니다.

挙句に
끝내, 결국 (~끝에 안 좋은 결과가 옴)(あげくに)
挙 (들 거) = 모든 것을 들어 올리다.
句 (글귀 구) = 문장이나 일의 마무리를 뜻함.
오랫동안 여러 시도를 하거나 끈질긴 과정 끝에 결국엔 좋지 않은 결말이나 꼴을 맞이하게 되었음을 나타내는 표현입니다.

掴む
잡다, 거머쥐다(つかむ)
掴 (잡을 획) = 扌 (손 수) + 国 (나라 국의 변형) : 손으로 물건이나 기회를 단단히 쥐어잡는다는 뜻입니다.
물건을 손으로 강하게 움켜쥐거나, 성공이나 기회, 핵심 내용을 자신의 것으로 확실하게 쥐는 것을 뜻하는 동사입니다.

苦労
고생(くろう)
苦 (쓸 고) = 艹 (풀 초) + 古 (옛 고) : 맛이 쓴 약초나 고통스러운 상태를 뜻합니다.
労 (일할 로) = 𤇾 (불꽃 로) + 力 (힘 력) : 힘을 들여 몸과 마음을 힘들게 수고스럽게 일한다는 뜻입니다.
몸과 마음을 힘들게 쏟아부으며 겪는 수고와 고통스러운 고생을 의미합니다.

さんざん
지긋지긋하게, 몹시(さんざん)
산산조각이 나거나 도가 지나쳐서 처참하고 몹시 심한 상태를 나타내는 부사입니다.

激しく
격렬하게(はげしく)
激 (격할 격) = 氵 (물 수) + 敫 (빛날 교) : 물살이 거세게 부딪히며 소리치듯 격렬하다는 뜻입니다.
움직임이나 변화, 감정 등이 보통을 넘어 매우 세차고 거칠게 일어나는 모양을 뜻하는 부사입니다.

我が家
우리 집(わがや)
我 (나 아) = 나 자신을 뜻함.
家 (집 가) = 사람이 모여 사는 집을 뜻함.
자신이 속해 있는 가정이나 자신이 살고 있는 집을 친근하게 이르는 말을 뜻합니다.

我が社
우리 회사(わがしゃ)
我 (나 아) = 우리를 뜻함.
社 (모일 사) = 직장이나 회사를 뜻함.
비즈니스 대화나 문장에서 자신이 소속되어 일하는 회사를 친근하고 소속감 있게 이르는 말입니다.

争う
다투다, 겨루다(あらそう)
争 (다툴 정) = 두 손으로 물건을 서로 잡아당기며 빼앗으려는 모양을 본뜬 상형자입니다.
이기기 위해 서로 말이나 행동, 힘으로 다투거나 경쟁하는 것을 뜻하는 동사입니다.

戦う
싸우다, 경쟁하다(たたかう)
戦 (싸울 전) = 무기를 들고 적과 부딪혀 싸우는 모습을 뜻합니다.
적이나 상대방에 맞서서 무력이나 실력을 겨루어 싸우거나 난관에 맞서는 것을 뜻하는 동사입니다.
