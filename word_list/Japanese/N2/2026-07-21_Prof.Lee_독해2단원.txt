목적
목적(もくてき)
目 (눈 목) = 사람의 눈 모양을 본뜬 글자로, 사물을 보고 목표를 정하는 기준을 뜻합니다.
的 (과녁 적) = 白 (흰 백) + 勺 (국자 작) : 화살이 뚫고 지나가는 하얗고 둥근 과녁의 중심을 뜻합니다.
눈(目)으로 똑바로 바라보고 나아가는 과녁(的)이라는 뜻으로, 실현하려고 하는 일의 지향점을 의미합니다.

日記
일기(にっき)
日 (날 일) = 태양의 모양을 본뜬 글자로, 날짜나 하루를 뜻합니다.
記 (기록할 기) = 言 (말씀 언) + 己 (몸 기) : 자기(己)의 생각이나 일어난 일을 말(言)로 적어 남긴다는 뜻입니다.
날마다(日) 있었던 일을 적어 기록(記)하는 글을 뜻합니다.

写真
사진(しゃしん)
写 (베낄 사) = 冖 (덮을 멱) + 与 (줄 여) : 물건을 덮어두고(冖) 모양을 그대로 옮겨 준다(与)는 뜻에서 베낀다는 의미가 되었습니다.
真 (참 진) = 匕 (변화할 화) + 目 (눈 목) + ∟ (숨을 은) + 八 (여덟 팔) : 눈(目)으로 직접 확인한 겉과 속이 같은 진짜 모습을 뜻합니다.
사물의 형태를 그대로 베껴서(写) 진짜(真) 모습대로 나타낸 결과물을 뜻합니다.

公開
공개(こうかい)
公 (공변될 공) = 八 (나눌 팔) + 厶 (사사로울 사) : 사사로운(厶) 것을 나누어(八) 여러 사람과 함께한다는 뜻입니다.
開 (열 개) = 門 (문 문) + 幵 (평평할 견) : 빗장을 풀고 문(門)을 활짝 열어젖힌다는 뜻입니다.
사사롭게 숨기지 않고 여럿이 볼 수 있도록 널리 열어(開) 놓는 것을 뜻합니다.

情報
정보(じょうほう)
情 (뜻 정/실정 정) = 忄 (마음 심) + 青 (푸를 청) : 마음(忄)속에 품은 푸르고(青) 맑은 감정이나 실제 사정을 뜻합니다.
報 (갚을 보/알릴 보) = 𠂡 (다스릴 집) + 卩 (무릎꿇을 절) + 又 (또 우) : 죄인을 다스려 죄 값을 치르게 하거나, 그 사실을 조정에 알린다는 뜻입니다.
실제 일어난 실정(情)을 상대방에게 알려(報) 주는 소식을 뜻합니다.

共有
공유(きょうゆう)
共 (한가지 공) = 廿 (스무 입) + 廾 (두손 받들 공) : 여러 사람이 두 손(廾)으로 물건을 함께 받쳐 든다는 뜻입니다.
有 (있을 유) = 𠂇 (왼손 좌) + 月 (고기 육) : 손(𠂇)에 고기(月)를 쥐고 있어 소유하고 있다는 뜻입니다.
두 사람 이상이 함께(共) 물건이나 정보를 나누어 가지고(有) 있는 것을 뜻합니다.

趣味
취미(しゅみ)
趣 (뜻 취/달릴 취) = 走 (달릴 주) + 取 (취할 취) : 마음이 쏠리는 곳을 취하기(取) 위해 달려간다(走)는 뜻입니다.
味 (맛 미) = 口 (입 구) + 未 (아닐 미) : 입(口)에 음식을 넣고 아직(未) 어떤 맛인지 음미한다는 뜻입니다.
마음이 쏠리는(趣) 대상을 즐기며 깊은 맛(味)을 느끼는 일을 뜻합니다.

友人
친구(ゆうじん)
友 (벗 우) = 𠂇 (왼손 좌) + 又 (오른손 우) : 두 사람이 서로 손(𠂇, 又)을 맞잡고 다정하게 돕는 벗을 뜻합니다.
人 (사람 인) = 두 사람이 서로 기대어 서 있는 모습을 본뜬 글자입니다.
다정하게 손을 맞잡고 지내는(友) 사람(人)을 뜻합니다.

長所
장점(ちょうしょ)
長 (길 장) = 머리카락이 긴 노인이 지팡이를 짚고 있는 모양으로, 길거나 뛰어나다는 뜻입니다.
所 (곳 소) = 戶 (지게문 호) + 斤 (도끼 근) = 도끼(斤)로 찍어 문(戶)을 짜서 정착해 사는 자리를 뜻합니다.
지닌 성질이나 능력 중 오래도록 길게(長) 빛나는 바(所)를 뜻합니다.

国籍
국적(こくせき)
国 (나라 국) = 囗 (에울 위) + 玉 (구슬 옥) = 담(囗)으로 둘러싸인 영토 안에서 보물(玉)을 지키며 다스리는 나라를 뜻합니다.
籍 (문서 적) = 竹 (대나무 죽) + 耒 (쟁기 뢰) + 昔 (옛 석) = 대나무(竹) 쪽에 쟁기질(耒)하듯 깊이 새겨 오래도록(昔) 보관하는 장부를 뜻합니다.
한 나라(国)의 국민으로서 등록된 공식적인 장부(籍)나 자격을 뜻합니다.

年齢
나이(ねんれい)
年 (해 년) = 禾 (벼 화) + 千 (일천 천) = 벼(禾)를 매년 수확하여 거두어들이는 햇수를 뜻합니다.
齢 (나이 령) = 齒 (이 치) + 令 (하여금 령) = 나이를 먹음에 따라 바뀌는 이(齒)의 상태로 나이를 가늠한다는 뜻입니다.
태어난 해(年)로부터 지나온 삶의 햇수와 나이(齢)를 의미합니다.

検索
검색(けんさく)
検 (검사할 검) = 木 (나무 목) + 斂 (거둘 렴의 생략형) : 나무(木) 상자 속에 든 문서들을 하나하나 들추어 조사한다는 뜻입니다.
索 (찾을 색/쓸쓸할 삭) = 𠂇 (손) + 幺 (실 요) + 糸 (실 사) : 꼬여 있는 실가닥(幺, 糸)을 손(𠂇)으로 더듬어가며 실마리를 찾는다는 뜻입니다.
기준에 따라 조사하고(検) 샅샅이 뒤져서 찾아내는(索) 행위를 뜻합니다.

利用者
이용자(りようしゃ)
利 (이로울 리) = 禾 (벼 화) + 刂 (칼 도) : 잘 드는 칼(刂)로 이삭(禾)을 수확하여 이익을 얻는다는 뜻입니다.
用 (쓸 용) = 나무로 짠 통이나 종의 모양을 본뜬 글자로, 도구를 가져다 쓴다는 뜻입니다.
者 (놈 자/사람 자) = 耂 (늙을 로) + 白 (말할 백) : 나이 든 어른(耂)이 아랫사람에게 타이르며 말(白)하는 '사람'을 뜻합니다.
이롭게(利) 도구나 서비스를 쓰는(用) 사람(者)을 뜻합니다.

再会
재회(さいかい)
再 (두 재) = 冂 (멀 경) + 冓 (쌓을 구) : 일이 한 번에 끝나지 않고 거듭하여 두 번 쌓인다는 뜻입니다.
会 (모일 회) = 亼 (모을 집) + 曾 (거듭 증의 생략형) : 뚜껑과 그릇이 맞물리듯 여러 사람이 한곳에 모인다는 뜻입니다.
헤어졌던 사람끼리 다시(再) 모여 만나는(会) 것을 뜻합니다.

交流
교류(こうりゅう)
交 (사귈 교) = 사람이 다리를 꼬고 앉아 있는 모양으로, 서로 엇갈리거나 왕래하며 사귄다는 뜻입니다.
流 (흐를 류) = 氵 (물 수) + 𠫓 (아기 나올 돌) + 川 (개천 천) : 양수가 터져 아기(𠫓)가 물(氵)처럼 흘러(川) 나온다는 뜻에서 흐르다라는 의미가 되었습니다.
서로 오가며 사귀고(交) 물처럼 자연스럽게 흘러 통함(流)을 뜻합니다.

結婚
결혼(けっこん)
結 (맺을 결) = 糸 (실 사) + 吉 (길할 길) : 혼인과 같은 길한(吉) 일을 위해 두 집안을 실(糸)로 단단히 묶어 인연을 맺는다는 뜻입니다.
婚 (혼인할 혼) = 女 (여자 여) + 昏 (어두울 혼) : 고대 사회에서 날이 어두워질(昏) 때 신부(女)를 맞이하는 의식을 치렀다는 데서 유래했습니다.
두 사람이 인연을 맺고(結) 부부가 되는 혼인(婚) 의식을 치르는 것을 뜻합니다.

欠点
결점(けってん)
欠 (하품할 흠/이지러질 결) = 사람이 입을 크게 벌리고 하품하는 모양으로, 기운이나 무언가가 부족하여 빠져 있다는 뜻입니다.
点 (점 점) = 占 (점 차지할 점) + 灬 (불 화) : 불(灬)을 피워 그을음으로 찍은 검은 점이나 흠집을 뜻합니다.
무엇인가 부족하여 빠진(欠) 부분이나 흠이 되는 점(点)을 뜻합니다.

個人情報
개인정보(こじんじょうほう)
個 (낱 개) = 人 (사람 인) + 固 (단단할 고) = 단단한 울타리(固) 안에 있는 사람(人) 한 명 한 명을 낱개로 센다는 뜻입니다.
人 (사람 인) = 두 사람이 서로 기대어 서 있는 모습을 본뜬 글자입니다.
情 (뜻 정/실정 정) = 忄 (마음 심) + 青 (푸를 청) : 마음(忄)속에 품은 푸르고(青) 맑은 감정이나 실제 사정을 뜻합니다.
報 (갚을 보/알릴 보) = 𠂡 (다스릴 집) + 卩 (무릎꿇을 절) + 又 (또 우) : 죄인을 다스려 죄 값을 치르게 하거나, 그 사실을 조정에 알린다는 뜻입니다.
낱낱의(個) 사람(人)에 관한 실제 사정(情)과 상세한 소식(報)을 뜻합니다.

流출
유출(りゅうしゅつ)
流 (흐를 류) = 氵 (물 수) + 𠫓 (아기 나올 돌) + 川 (개천 천) : 물이 개천을 따라 흘러간다는 뜻입니다.
出 (날 출) = 凵 (입 벌릴 감) + 屮 (싹 날 철) : 구덩이나 문밖으로 싹이 돋아 밖으로 나온다는 뜻입니다.
안쪽에 있어야 할 정보나 물질이 밖으로 흘러(流) 나가(出) 버리는 것을 뜻합니다.

事件
사건(じけん)
事 (일 사) = 깃발을 단 깃대를 손으로 붙잡고 있는 모양으로, 기록하거나 처리해야 할 행정적인 일을 뜻합니다.
件 (물건 건/사건 건) = 人 (사람 인) + 牛 (소 우) : 사람이 커다란 소(牛)를 부위별로 나누어 한 건, 한 건 센다는 뜻입니다.
사회적으로 문제를 일으킨 일(事)의 한 건(件) 한 건을 뜻합니다.

警告
경고(けいこく)
警 (깨우칠 경) = 敬 (공경할 경) + 言 (말씀 언) : 흐트러진 마음을 바로잡도록 공경하고 엄숙한(敬) 말(言)로 일깨운다는 뜻입니다.
告 (알릴 고) = 牛 (소 우) + 口 (입 구) : 신에게 제물로 바칠 소(牛)를 앞에 두고 입(口)으로 축문을 읊어 고한다는 뜻입니다.
잘못을 저지르지 않도록 엄하게 타이르고(警) 알려(告) 주의를 주는 것을 뜻합니다.

犯罪
범죄(はんざい)
犯 (범할 범) = 犭 (개 사슴록 변) + 㔾 (병부 절) : 사나운 짐승(犭)이 사람을 덮쳐 법이나 경계를 들이받고 넘는다는 뜻입니다.
罪 (허물 죄) = 网 (그물 망) + 非 (아닐 비) : 그릇된(非) 짓을 한 사람을 그물(网)로 잡아 처벌하는 허물을 뜻합니다.
법을 위반하여 범하고(犯) 지은 잘못이나 죄(罪)를 뜻합니다.

増加
증가(ぞうか)
増 (더할 증) = 土 (흙 토) + 曾 (거듭 증) : 흙(土)을 거듭거듭(曾) 쌓아 올려 양을 늘린다는 뜻입니다.
加 (더할 가) = 力 (힘 력) + 口 (입 구) : 말(口)로 응원하며 힘(力)을 보태어 더해준다는 뜻입니다.
수량이나 부피가 거듭 쌓여 늘어나고(増) 더해지는(加) 것을 뜻합니다.

原因
원인(げんいん)
原 (근원 원) = 厂 (민엄호/기슭) + 泉 (샘 천) : 산기슭(厂) 바위 틈에서 맑은 샘물(泉)이 솟아나오는 시작점을 뜻합니다.
因 (인할 인) = 囗 (에울 위) + 大 (큰 대) : 사람이 돗자리(大) 위에 누워 있는 모양으로, 어떤 일의 바탕이나 연유를 뜻합니다.
어떤 사태가 빚어지게 된 근본적인 시작점(原)과 바탕(因)을 뜻합니다.

心理
심리(しん리)
心 (마음 심) = 사람의 심장 모양을 본뜬 글자로, 생각이나 마음을 뜻합니다.
理 (다스릴 리/이치 리) = 王 (임금 왕/구슬 옥) + 里 (마을 리) : 거친 원석(옥)을 다듬어 결(里)을 살려낸다는 뜻에서 우주의 이치나 법칙을 의미합니다.
사람의 마음(心)이 작용하는 이치(理)나 의식 상태를 뜻합니다.

責任感
책임감(せきにんかん)
責 (꾸짖을 책/맡을 책) = 𢂔 (가시나무) + 貝 (조개 패) : 빚을 진 사람에게 재물(貝)을 갚으라고 가시 돋친 말로 다그치며 책무를 지운다는 뜻입니다.
任 (맡길 임) = 人 (사람 인) + 壬 (아홉째 천간 임/짊어질 임) : 사람(人)이 짐을 등에 메고(壬) 있듯이 임무를 맡는다는 뜻입니다.
感 (느낄 감) = 咸 (다 함) + 心 (마음 심) : 주변의 자극에 온 마음(心)이 다(咸) 감응하여 감정을 느낀다는 뜻입니다.
맡아서 해야 할 임무(責)와 직책(任)을 소중히 여기고 느끼는(感) 마음을 뜻합니다.
