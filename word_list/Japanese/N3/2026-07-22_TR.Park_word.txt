建
세울 건(건/켄/타테루)
建 (세울 건) = 廴 (길게 걸을 인) + 聿 (붓 율) : 손에 붓(聿)을 들고 길게 걸어가며(廴) 설계도를 그리고 건물을 세운다는 뜻입니다.
건물을 짓거나 법률 등을 세우는 것을 의미하는 단일 한자어입니다.

健
건강할 건(건/켄/스코야카)
健 (굳셀 건) = 亻 (사람 인) + 建 (세울 건) : 사람(亻)이 반듯하게 세워진(建) 건물처럼 몸이 튼튼하고 굳세다는 뜻입니다.
몸이나 정신이 병 없이 튼튼하고 바른 상태를 의미하는 단일 한자어입니다.

鍵
열쇠 건(건/켄/카기)
鍵 (열쇠 건) = 金 (쇠 금) + 建 (세울 건) : 쇠(金)로 만들어져 문을 굳게 세워(建) 잠그거나 여는 도구라는 뜻입니다.
자물쇠를 열고 잠그는 연장이나 문제를 해결하는 실마리를 뜻하는 단일 한자어입니다.

平地
평지(へいち)
平 (평평할 평) = 干 (방패 간) + 八 (여덟 팔) : 방패(干)처럼 판판하게 고르고(八) 치우침 없이 평평하다는 뜻입니다.
地 (땅 지) = 土 (흙 토) + 也 (어조사 야/뱀) : 뱀(也)이 기어 다니는 흙(土) 바닥을 의미하여 땅을 뜻합니다.
높낮이가 없이 평평하고(平) 고른 땅(地)을 의미합니다.

壁
벽(かべ)
壁 (벽 벽) = 辟 (피할 피/임금 벽) + 土 (흙 토) : 바람이나 적을 피하기(辟) 위해 흙(土)을 높이 쌓아 올린 구조물이라는 뜻입니다.
방이나 공간의 둘레를 막아 세운 흙이나 벽돌 구조물을 뜻하는 단일 한자어입니다.

紙
종이(かみ)
紙 (종이 지) = 糸 (실 사) + 氏 (성 씨/각시 씨) : 실(糸)처럼 얇은 섬유를 펴서 면을 고르게(氏) 엉기게 하여 만든 물건이라는 뜻입니다.
글을 쓰거나 물건을 싸는 데 쓰는 얇은 물건을 뜻하는 단일 한자어입니다.

社
모일 사(사/샤/야시로)
社 (모일 사) = 礻 (보일 시) + 土 (흙 토) : 흙(土)으로 만든 제단(礻)에 마을 사람들이 모이는 모습에서 공동체나 조직을 의미합니다.
사람들이 특정한 목적을 위해 한곳에 모인 집단이나 회사를 뜻하는 단일 한자어입니다.

神
귀신 신(신/카미)
神 (귀신 신) = 礻 (보일 시) + 申 (펼 신) : 제단(礻)에서 기도를 올리면 번개가 퍼지듯(申) 영험한 기운이 나타나는 초자연적인 존재를 뜻합니다.
인간의 능력을 뛰어넘는 절대자나 신령스러운 기운을 뜻하는 단일 한자어입니다.

祝賀
축하(しゅくが)
祝 (빌 축) = 礻 (보일 시) + 兄 (맏 형) : 맏형(兄)이 제단(礻) 앞에서 집안을 대표하여 신에게 복을 빈다는 뜻입니다.
賀 (하례할 하) = 加 (더할 가) + 貝 (조개 패) : 좋은 일에 돈이나 재물(貝)을 더하여(加) 보태며 기뻐하고 축하한다는 뜻입니다.
남의 좋은 일에 대해 복을 빌고(祝) 예물을 더하여 기뻐하는(賀) 것을 의미합니다.

祈り
기도(いのり)
祈 (빌 기) = 礻 (보일 시) + 斤 (도끼 근) : 도끼(斤)로 희생양을 잡아 제단(礻)에 올리고 바라는 것을 빈다는 뜻입니다.
신이나 절대자에게 바라는 바가 이루어지기를 간절히 비는 행위를 뜻하는 명사입니다.

失礼
실례(しつれい)
失 (잃을 실) = 手 (손 수) + 乙 (새 을) : 손(手)에 쥐고 있던 새(乙)를 날려 보내 잃어버렸다는 뜻입니다.
礼 (예절 례) = 礻 (보일 시) + 乚 (새 을) : 신에게 제사를 지낼 때 갖추는 몸가짐에서 예절을 뜻합니다.
상대방에게 마땅히 지켜야 할 예의나 예절(礼)을 잃어버리고(失) 어긋나게 행동하는 것을 의미합니다.

髪
머리카락(かみ)
髪 (터럭 발) = 髟 (터럭 표) + 抜 (뽑을 발의 변형) : 머리털(髟)이 길게 뻗어 나온 모양을 뜻합니다.
사람의 머리에 자라는 털이나 머리카락을 뜻하는 단일 한자어입니다.

空
하늘, 비다(そら, あく)
空 (빌 공) = 穴 (구멍 혈) + 工 (장인 공) : 장인(工)이 흙이나 돌을 파내어 둥근 구멍(穴)을 만들어 속이 텅 비었다는 뜻입니다.
머리 위에 넓게 열린 하늘을 뜻하거나, 속이 텅 비어 아무것도 없는 상태를 뜻하는 단일 한자어입니다.

体
몸(からだ)
体 (몸 체) = 亻 (사람 인) + 本 (근본 본) : 사람(亻)의 삶과 생명을 지탱하는 근본(本)이 되는 뼈대와 고기라는 뜻의 신자체입니다.
사람이나 동물의 형상을 이루는 전체 몸통이나 형태를 뜻하는 단일 한자어입니다.

植える
심다(うえる)
植 (심을 식) = 木 (나무 목) + 直 (곧을 직) : 나무(木)가 쓰러지지 않게 땅에 곧고(直) 바르게 세워 심는다는 뜻입니다.
식물이 자라도록 흙 속에 뿌리를 묻어 주거나, 어떤 사상이나 마음을 뿌리내리게 하는 행위를 뜻하는 동사입니다.

直す
고치다(なおす)
直 (곧을 직) = 十 (열 십) + 目 (눈 목) + 乚 (숨을 은) : 열(十) 개의 눈(目)으로 숨은(乚) 곳까지 곧게 꿰뚫어 본다는 뜻입니다.
어긋나거나 잘못된 것을 원래의 곧고(直) 바른 상태로 다시 되돌려 고치는 행위를 뜻하는 동사입니다.

直接
직접(ちょくせつ)
直 (곧을 직) = 굽거나 치우침 없이 곧고 바르다는 뜻입니다.
接 (이을 접) = 扌 (손 수) + 妾 (가까이 할 첩) : 손을 뻗어 가까이 맞닿고 잇는다는 뜻입니다.
중간에 다른 사람이나 사물을 거치지 않고 곧바로(直) 닿아서 이어지는(接) 것을 의미합니다.

値
값 치(치/아타이)
値 (값 치) = 亻 (사람 인) + 直 (곧을 직) : 사람(亻)이 물건의 곧고(直) 합당한 가치를 매긴다는 뜻입니다.
물건이나 노력에 대해 매겨진 가치나 가격을 뜻하는 단일 한자어입니다.

動く
움직이다(うごく)
動 (움직일 동) = 重 (무거울 중) + 力 (힘 력) : 무거운(重) 짐을 힘(力)을 주어 옮기며 움직인다는 뜻입니다.
어떤 자리에 있던 사물이나 생물이 다른 자리로 옮겨가거나 자세를 바꾸는 것을 뜻하는 동사입니다.

疑う
의심하다(うたがう)
疑 (의심할 의) = 여러 무기 앞에서 발을 멈추고 헷갈려 하는 모습에서 의심하고 머뭇거린다는 뜻입니다.
어떤 사실에 대해 믿지 못하고 헷갈려 하며 수상하게 여기는 것을 뜻하는 동사입니다.

打つ
치다, 때리다(うつ)
打 (칠 타) = 扌 (손 수) + 丁 (고무래 정/못 정) : 손(扌)에 연장을 들고 못(丁)을 내리쳐 박는다는 뜻입니다.
손이나 도구로 물건을 강하게 내리치거나 때리는 행위를 뜻하는 동사입니다.

写す
베끼다(うつす)
写 (베낄 사) = 冖 (덮을 멱) + 与 (줄 여) : 지붕(冖) 아래에서 물건을 이리저리(与) 고스란히 옮겨 놓듯 있는 그대로 그려낸다는 뜻의 신자체입니다.
글이나 그림, 사진 등을 원본과 똑같은 모양으로 다른 곳에 옮겨 담는 것을 뜻하는 동사입니다.

映す
비추다(うつす)
映 (비칠 영) = 日 (날 일/해) + 央 (가운데 앙) : 해(日)가 하늘 가운데(央)에 떠서 밝은 빛을 사방에 비춘다는 뜻입니다.
물이나 거울, 화면 등에 어떤 사물의 그림자나 형태가 나타나도록 비추는 것을 뜻하는 동사입니다.

移す
옮기다(うつす)
移 (옮길 이) = 禾 (벼 화) + 多 (많을 다) : 벼(禾) 모종을 많고(多) 빽빽한 곳에서 빈 곳으로 옮겨 심는다는 뜻입니다.
원래 있던 자리나 상태에서 다른 위치나 상태로 자리를 옮기는 행위를 뜻하는 동사입니다.

生まれる
태어나다(うまれる)
生 (날 생) = 땅 위로 초목의 싹이 돋아나는 모양을 본뜬 상형자로, 세상에 나오다, 태어나다는 뜻입니다.
세상에 생명을 가지고 처음으로 싹트듯 나오거나, 새로운 것이 만들어지는 것을 뜻하는 동사입니다.

売る
팔다(うる)
売 (팔 매) = 士 (선비 사) + 冖 (덮을 멱) + 儿 (어진사람 인)의 신자체 : 본래는 물건을 밖으로 내어놓고 다른 사람에게 넘겨준다는 뜻입니다.
자신이 가진 물건이나 권리를 남에게 주고 그에 해당하는 돈이나 대가를 받는 행위를 뜻하는 동사입니다.

売れる
팔리다(うれる)
売 (팔 매) = 물건을 대가를 받고 남에게 넘겨준다는 뜻입니다.
자신이 내놓은 물건이 다른 사람에게 사들여져 밖으로 잘 나가는 상태를 뜻하는 동사입니다.

売られる
팔리다(うられる)
売 (팔 매) = 물건을 대가를 받고 남에게 넘겨준다는 뜻입니다.
売る(팔다)의 수동태 표현으로, 다른 사람에 의해 내 물건이나 권리가 남에게 넘겨지는 상태를 뜻하는 표현입니다.

得る
얻다(える)
得 (얻을 득) = 彳 (쉬엄쉬엄 갈 척) + 貝 (조개 패) + 寸 (마디 촌/손) : 길을 가다가(彳) 재물(貝)을 손(寸)으로 얻게 되었다는 뜻입니다.
노력이나 과정을 통해 자신에게 필요한 것을 구하여 손에 넣는 행위를 의미하는 동사입니다.

異なる
다르다(ことなる)
異 (다를 이) = 畀 (줄 비) + 廾 (두 손 공)의 변형으로, 귀신이나 괴물이 가면을 쓴 모습에서 본래와 다르고 기괴하다는 뜻이 되었습니다.
서로 비교했을 때 같지 않고 상태나 모양이 다른 것을 의미하는 동사입니다.

叱られる
혼나다, 꾸중 듣다(しかられる)
叱 (꾸짖을 질) = 口 (입 구) + 七 (일곱 칠/비수 비) : 입(口) 밖으로 비수(七)처럼 날카로운 말을 내뱉어 꾸짖는다는 뜻입니다.
叱る(꾸짖다)의 수동태 표현으로, 다른 사람에게 날카로운 지적이나 꾸중을 듣게 되는 상태를 뜻하는 표현입니다.

送る
보내다(おくる)
送 (보낼 송) = 辶 (책받침) + 灷 (불 화의 변형) : 물건을 들고 길을 따라(辶) 다른 사람에게 가져다준다는 뜻입니다.
물건을 다른 곳으로 부치거나, 사람을 배웅하거나, 시간을 보내는 행위를 뜻하는 동사입니다.

違い
차이, 다름(ちがい)
違 (어긋날 위) = 辶 (책받침) + 韋 (가죽 위/어긋날 위) : 두 사람이 등을 돌려 길(辶)을 서로 어긋나게 반대로 간다는 뜻입니다.
서로 어긋나서 같지 않고 차이가 나는 상태를 뜻하는 명사입니다.

返す
돌려주다(かえす)
返 (돌이킬 반) = 辶 (책받침) + 反 (돌이킬 반) : 길을 가다가(辶) 발길을 돌려(反) 반대 방향으로 되돌아온다는 뜻입니다.
빌린 물건이나 받았던 것을 본래의 주인에게 다시 되돌려 주는 행위를 뜻하는 동사입니다.

追う
쫓다(おう)
追 (쫓을 추) = 辶 (책받침) + 𠂤 (언덕 퇴/무리) : 앞서가는 무리(𠂤)의 뒤를 따라 길을(辶) 뒤쫓아간다는 뜻입니다.
앞서가는 사람이나 사물의 뒤를 따라서 쫓아가거나, 목표를 향해 나아가는 행위를 뜻하는 동사입니다.

知れる
알려지다, 판명되다(しれる)
知 (알 지) = 矢 (화살 시) + 口 (입 구) : 화살(矢)처럼 빠르게 입(口)으로 말할 수 있을 만큼 사물을 명확히 안다는 뜻입니다.
숨겨져 있던 사실이나 사물의 상태가 남들에게 명확하게 알려지고 밝혀지는 상태를 뜻하는 동사입니다.

選ぶ
고르다(えらぶ)
選 (가릴 선) = 辶 (책받침) + 巽 (유순할 손/가릴 손) : 여럿이 모인 길(辶)에서 좋은 것을 이리저리 살펴보고 가려낸다는 뜻입니다.
여러 개 중에서 자신의 목적이나 기준에 맞는 것을 가려내고 고르는 행위를 뜻하는 동사입니다.

怒る
화내다(おこる)
怒 (성낼 노) = 奴 (종 노) + 心 (마음 심) : 종(奴)처럼 억눌렸던 감정이 마음(心)속에서 치밀어 올라 성을 낸다는 뜻입니다.
마음속에서 불만이 치밀어 올라 분노하고 화를 내는 감정 상태를 뜻하는 동사입니다.

落ちる
떨어지다(おちる)
落 (떨어질 락) = 艹 (풀 초) + 洛 (강 이름 락) : 풀(艹)이나 잎이 물(洛) 위로 떨어져 흘러간다는 뜻입니다.
위에 있던 물건이 아래로 떨어지거나, 시험 등에서 탈락하여 떨어지는 것을 뜻하는 동사입니다.

落ち着く
진정하다, 차분해지다(おちつく)
落 (떨어질 락) = 공중에 떠 있던 것이 아래로 떨어져 내려온다는 뜻입니다.
着 (붙을 착) = 羊 (양 양) + 目 (눈 목)의 변형으로, 옷을 몸에 딱 맞게 입어 밀착한다는 뜻입니다.
들떠 있던 마음이나 사물이 제자리로 떨어져(落) 밀착하듯(着) 고정되어 차분해지는 상태를 뜻하는 복합 동사입니다.

驚く
놀라다(おどろく)
驚 (놀랄 경) = 敬 (공경할 경) + 馬 (말 마) : 말(馬)이 채찍질 소리에 흠칫 놀라 펄쩍 뛰는 모습에서 놀라다는 뜻이 되었습니다.
갑작스러운 일이나 예기치 못한 상황을 겪고 말이 펄쩍 뛰듯 놀라는 감정 상태를 뜻하는 동사입니다.

覚える
기억하다, 배우다(おぼえる)
覚 (깨달을 각) = 学 (배울 학)의 윗부분 + 見 (볼 견) : 가르침을 받고 눈(見)으로 보아 사물의 이치를 깨달아 안다는 뜻의 신자체입니다.
어떤 사실을 머릿속에 기억해 두거나, 새로운 지식이나 감각을 몸으로 익혀 깨닫는 것을 뜻하는 동사입니다.

危ない
위험하다(あぶない)
危 (위태할 위) = 厄 (재앙 액) + 㔾 (병부 절) : 벼랑 위에서 사람(㔾)이 재앙(厄)을 만나 굴러떨어질 듯 아슬아슬하다는 뜻입니다.
다치거나 해를 입을 가능성이 있어서 안심할 수 없고 아슬아슬하게 위험한 상태를 뜻하는 형용사입니다.

怪しい
수상하다, 의심스럽다(あやしい)
怪 (기이할 괴) = 忄 (마음 심) + 圣 (힘쓸 골/땅 신) : 마음(忄)으로 이해하기 어려운 기이하고 이상한(圣) 것을 뜻합니다.
평소와 다르거나 기이하여 왠지 의심스럽고 수상하게 느껴지는 성질을 뜻하는 형용사입니다.

洗い
씻기, 세탁(あらい)
洗 (씻을 세) = 氵 (물 수) + 先 (먼저 선) : 발을 내밀고(先) 물(氵)로 깨끗하게 씻어낸다는 뜻입니다.
물과 같은 액체로 더러운 것을 닦아내거나 씻어내는 행위 그 자체를 뜻하는 명사입니다.

薄い
얇다, 묽다(うすい)
薄 (엷을 박) = 艹 (풀 초) + 溥 (넓을 보의 변형) : 풀(艹)이 무성하게 자라 엉킨 모습에서 점차 두께가 얇고 넓게 펴진다는 뜻이 파생되었습니다.
두께가 두껍지 않고 얇거나, 맛이나 색이 진하지 않고 묽은 성질을 뜻하는 형용사입니다.

美しい
아름답다(うつくしい)
美 (아름다울 미) = 羊 (양 양) + 大 (클 대) : 살이 통통하게 크고(大) 훌륭한 양(羊)의 모습에서 보기 좋고 아름답다는 뜻이 되었습니다.
모양이나 마음씨 등이 보기 좋고 훌륭하여 아름답게 느껴지는 성질을 뜻하는 형용사입니다.

価
값 가(네/아타이 - ち)
価 (값 가) = 亻 (사람 인) + 賈 (장사할 고의 생략형) : 사람(亻)이 장사를 하며 물건에 가치나 가격을 매긴다는 뜻의 신자체입니다.
물건이나 노력에 대해 매겨진 화폐의 액수나 그 가치를 뜻하는 단일 한자어입니다.

過
지날 과(과/스기루)
過 (지날 과) = 辶 (책받침) + 咼 (비뚤어질 과) : 길을 가다가(辶) 어긋나듯(咼) 바른 위치나 정도를 벗어나 지나쳐버린다는 뜻입니다.
시간이 지나가거나 어떤 기준을 넘어서버린 상태를 뜻하는 단일 한자어입니다.

活
살 활(활/이키루)
活 (살 활) = 氵 (물 수) + 舌 (혀 설) : 물(氵)이 거침없이 흐르듯 혀(舌)가 유연하게 움직여 생기가 넘친다는 뜻입니다.
생명이 있어 살아 움직이거나, 생기와 활력이 넘치는 상태를 뜻하는 단일 한자어입니다.

感
느낄 감(감/칸지루)
感 (느낄 감) = 咸 (다 함) + 心 (마음 심) : 어떤 자극을 받아 마음(心)속에 모든(咸) 감정이 가득 차오르고 움직인다는 뜻입니다.
외부의 자극을 받아 마음이 감동하거나 오감으로 무언가를 깨달아 느끼는 것을 뜻하는 단일 한자어입니다.

観
볼 관(관/미루)
観 (볼 관) = 雚 (황새 관) + 見 (볼 견) : 황새(雚)처럼 눈을 크게 뜨고 자세히 두루 살펴본다(見)는 뜻입니다.
눈을 크게 뜨고 대상의 모습이나 풍경을 꼼꼼히 살피고 바라보는 것을 뜻하는 단일 한자어입니다.

社
모일 사(사/샤/야시로)
社 (모일 사) = 礻 (보일 시) + 土 (흙 토) : 흙(土)으로 만든 제단(礻)에 마을 사람들이 모이는 모습에서 공동체나 조직을 의미합니다.
사람들이 특정한 목적을 위해 한곳에 모인 집단이나 회사를 뜻하는 단일 한자어입니다.

建
세울 건(건/켄/타테루)
建 (세울 건) = 廴 (길게 걸을 인) + 聿 (붓 율) : 손에 붓(聿)을 들고 길게 걸어가며(廴) 설계도를 그리고 건물을 세운다는 뜻입니다.
건물을 짓거나 법률 등을 세우는 것을 의미하는 단일 한자어입니다.

健
건강할 건(건/켄/스코야카)
健 (굳셀 건) = 亻 (사람 인) + 建 (세울 건) : 사람(亻)이 반듯하게 세워진(建) 건물처럼 몸이 튼튼하고 굳세다는 뜻입니다.
몸이나 정신이 병 없이 튼튼하고 바른 상태를 의미하는 단일 한자어입니다.

鍵
열쇠 건(건/켄/카기)
鍵 (열쇠 건) = 金 (쇠 금) + 建 (세울 건) : 쇠(金)로 만들어져 문을 굳게 세워(建) 잠그거나 여는 도구라는 뜻입니다.
자물쇠를 열고 잠그는 연장이나 문제를 해결하는 실마리를 뜻하는 단일 한자어입니다.

大人
어른(おとな)
大 (클 대) = 사람이 팔다리를 크게 벌리고 있는 모습에서 덩치가 크다는 뜻입니다.
人 (사람 인) = 두 발로 서 있는 사람의 옆모습을 본뜬 상형자입니다.
덩치가 크게(大) 자라나 신체적, 정신적으로 성숙한 사람(人)을 뜻합니다.

血
피(ち)
血 (피 혈) = 皿 (그릇 명) + 丶 (불똥 주) : 제사용 그릇(皿)에 희생양의 붉은 피(丶)가 담겨 있는 모양을 본뜬 상형자입니다.
동물이나 사람의 몸속 혈관을 타고 흐르는 붉은색의 액체를 뜻하는 단일 한자어입니다.

皿
접시(さら)
皿 (그릇 명) = 음식을 담기 위해 속이 얕게 파이고 넓적하게 만들어진 그릇의 모양을 본뜬 상형자입니다.
음식을 담아내는 넓적하고 오목한 그릇이나 접시를 뜻하는 단일 한자어입니다.

祈る
기도하다(いのる)
祈 (빌 기) = 礻 (보일 시) + 斤 (도끼 근) : 도끼(斤)로 희생양을 잡아 제단(礻)에 올리고 바라는 바를 빈다는 뜻입니다.
신이나 절대자에게 바라는 바가 이루어지기를 간절히 비는 행위를 뜻하는 동사입니다.

失礼
실례(しつれい)
失 (잃을 실) = 手 (손 수) + 乙 (새 을) : 손(手)에 쥐고 있던 새(乙)를 날려 보내 잃어버렸다는 뜻입니다.
礼 (예절 례) = 礻 (보일 시) + 乚 (새 을) : 신에게 제사를 지낼 때 갖추는 몸가짐에서 예절을 뜻합니다.
마땅히 지켜야 할 예의나 예절(礼)을 잃어버리고(失) 어긋나게 행동하는 것을 의미합니다.

受かる
합격하다, 붙다(うかる)
受 (받을 수) = 爫 (손톱 조/손) + 冖 (덮을 멱) + 又 (또 우/손) : 위에서 손(爫)으로 물건을 주고 아래에서 손(又)으로 받는 모양에서 받다라는 뜻이 되었습니다.
시험이나 검사 등에서 요구하는 기준을 받아들여 거머쥐고 합격하는 것을 뜻하는 동사입니다.

受ける
받다, 시험을 치다(うける)
受 (받을 수) = 爫 (손톱 조/손) + 冖 (덮을 멱) + 又 (또 우/손) : 손에서 손으로 물건을 건네받는 모습입니다.
타인이 주는 물건이나 자극, 영향 등을 받아들이거나 시험에 응시하여 치르는 행위를 뜻하는 동사입니다.

値段
가격, 값(ねだん)
値 (값 치) = 亻 (사람 인) + 直 (곧을 직) : 사람(亻)이 물건의 곧고(直) 합당한 가치를 매긴 값이라는 뜻입니다.
段 (구빌 단) = 殳 (몽둥이 수) + 叚 (빌릴 가) : 비탈길을 다져 계단처럼 층계를 만든다는 뜻에서 등급이나 구분을 뜻합니다.
물건의 가치에 따라 등급별로 매겨진(値) 화폐의 액수나 값(段)을 의미합니다.

通じる
통하다, 알리다(つうじる)
通 (통할 통) = 辶 (책받침) + 甬 (길 용) : 막힘없이 길(甬)을 따라 나아간다(辶)는 뜻입니다.
막힘없이 서로 이어져 뜻이나 말이 전달되거나, 길이나 교통이 서로 연결되는 것을 뜻하는 동사입니다.

通う
다니다, 왕래하다(かよう)
通 (통할 통) = 辶 (책받침) + 甬 (길 용) : 길을 따라 막힘없이 왕래한다는 뜻입니다.
학교나 직장 등 일정한 목적지를 일정한 간격을 두고 계속해서 오가는 행위를 뜻하는 동사입니다.

寄る
들르다, 가까이 가다(よる) - 제시하신 오타 よおる의 바른 표현
寄 (붙일 기) = 宀 (집 엄) + 奇 (기이할 기) : 집(宀)에 잠시 머물거나 의지하여 붙어 있다는 뜻입니다.
길을 가다가 목적지로 가는 도중에 다른 곳에 잠시 들르거나, 사물에 가까이 바짝 접근하는 행위를 뜻하는 동사입니다.

寄せる
의지하다, 가까이 대다, 모으다(よせる) - 寄る와 함께 자주 쓰이는 관련 단어
寄 (붙일 기) = 宀 (집 엄) + 奇 (기이할 기) : 잠시 들르거나 한곳에 붙여놓는다는 뜻입니다.
물건이나 마음을 한곳으로 바짝 가까이 대거나 모아들이는 행위를 뜻하는 동사입니다.
