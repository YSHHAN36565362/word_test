開発部
개발부(かいはつぶ)
開 (열 개) = 門 (문 문) + 幵 (평평할 견) : 닫힌 문(門)을 두 손으로 평평하게(幵) 밀어 연다는 뜻입니다.
発 (필 발) = 弓 (활 궁) + 殳 (몽둥이 수) + 癶 (등질 발) : 활(弓)을 쏘아 화살이 밖으로 뻗어나간다는 뜻입니다.
部 (떼 부) = 咅 (침 부/나눌 부) + 阝 (고을 읍) : 침을 튀기며 말을 나누듯(咅) 고을(阝)이나 무리를 나눈다는 뜻으로, 조직의 한 단위를 의미합니다.
새로운 것을 열어(開) 발전시키고(発) 만들어 내는 업무를 담당하는 부서(部)를 뜻합니다.

総務
총무(そうむ)

務 (힘쓸 무) = 敄 (힘쓸 무) + 力 (힘 력) : 창을 들고 치며(敄) 힘(力)을 쏟아 맡은 바 임무를 다한다는 뜻입니다.
조직 내의 여러 갈래로 나뉜 전반적인(総) 업무나 일(務)을 통합하여 관리하는 직무를 뜻합니다.

勤怠
근무태도(きんたい)

怠 (게으를 태) = 台 (별 태/기뻐할 태) + 心 (마음 심) : 마음(心)이 풀어져 편안함(台)만을 추구하여 게으르다는 뜻입니다.
직장에 부지런히 출근하여 일하는 상태(勤)와 게으름을 피워 결근이나 지각을 하는 상태(怠)를 아울러 이르는 말로, 출결 상황을 의미합니다.

就業規則
취업규칙(しゅうぎょうきそく)
就 (나아갈 취) = 京 (서울 경) + 尤 (더욱 우) : 높은 곳(京)을 향해 더욱(尤) 나아간다는 뜻입니다.
業 (업 업) = 丵 (풀무성할 착) + 木 (나무 목) : 정교하게 조각된 나무(木) 받침대 위에서 정성 들여 하는 직업을 뜻합니다.
規 (법 규) = 夫 (지아비 부) + 見 (볼 견) : 목수가 잣대(夫)를 눈(見)으로 보며 기준을 맞춘다는 뜻입니다.
則 (법칙 칙) = 貝 (조개 패) + 刂 (칼 도) : 재물(貝)을 칼(刂)로 공평하게 나누는 법칙이라는 뜻입니다.
직업에 나아가(就) 일(業)을 할 때 지켜야 하는 기준(規)과 법칙(則)을 의미합니다.

남의 뒤를 따르거나, 정해진 규칙이나 명령에 순응하여 그대로 행동한다는 뜻입니다.

勤 (부지런할 근) = 堇 (진흙 근) + 力 (힘 력) : 힘을 다해 부지런히 일함(근무)을 뜻합니다.

休 (쉴 휴) = 亻 (사람 인) + 木 (나무 목) : 사람(亻)이 나무(木) 그늘에 기대어 쉰다는 뜻입니다.
밝은 낮(昼) 시간대에 일을 잠시 멈추고 쉬는(休) 것으로, 직장에서의 중식 휴게 시간을 의미합니다.

어떤 기준점(以)을 포함하여 그 바깥(外)의 범위나 다른 대상들을 가리킬 때 사용하는 단어입니다.

머신, 기계(マシン)
Machine (머신) = 기계 장치를 뜻하는 영어 단어에서 유래했습니다.
컴퓨터, 서버, 혹은 복잡한 부품으로 이루어져 특정한 동력이나 연산을 수행하는 하드웨어 장비를 의미합니다.

セットアップ
세트업(セットアップ)
Setup (셋업) = 설치하다, 준비하다라는 뜻의 영어 단어에서 유래했습니다.
새로운 컴퓨터(머신)나 소프트웨어를 처음 도입했을 때, 사용자가 즉시 구동할 수 있도록 환경을 설정하고 준비하는 작업을 의미합니다.
