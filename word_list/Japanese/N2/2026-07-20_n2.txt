燃料
연료(ねんりょう)
燃 (탈 연) = 火 (불 화) + 然 (그러할 연) : 然은 고기를 개가 불에 구워 먹는 모습에서 타다라는 의미가 파생되었습니다. 불(火)이 타오르는 모습을 뜻합니다.
料 (헤아릴 료) = 米 (쌀 미) + 斗 (말 두) : 쌀(米)을 말(斗)로 헤아려 분량을 잰다는 뜻에서 재료나 요금을 의미합니다.
불에 타오르는(燃) 재료(料)를 의미합니다.

愛情
애정(あいじょう)
愛 (사랑 애) = 爪 (손톱 조) + 冖 (덮을 멱) + 心 (마음 심) + 夂 (천천히 걸을 쇠) : 무언가를 가슴(心)에 품고 천천히 걸어가며(夂) 아끼는 마음에서 사랑을 뜻합니다.
情 (뜻 정) = 忄 (마음 심) + 青 (푸를 청) : 마음(忄)속 깊은 곳에서 우러나오는 푸른(青) 생명력처럼 순수하고 참된 감정을 의미합니다.
대상을 아끼고 사랑하는(愛) 순수한 마음과 감정(情)을 뜻합니다.

委員
위원(いいん)
委 (맡길 위) = 禾 (벼 화) + 女 (계집 녀) : 벼(禾) 이삭이 늘어지듯 여자(女)가 다소곳이 순종하는 모습에서 일을 맡기다는 뜻이 파생되었습니다.
員 (인원 원) = 口 (입 구) + 貝 (조개 패) : 솥 위에 둥근 입구(口)가 있는 모습에서 수효나 사람을 세는 단위를 뜻합니다.
특정한 일을 맡아서(委) 처리하는 사람(員)을 의미합니다.

意識
의식(いしき)
意 (뜻 의) = 音 (소리 음) + 心 (마음 심) : 마음(心)속의 소리(音), 즉 생각이나 뜻을 의미합니다.
識 (알 식) = 言 (말씀 언) + 戠 (찰흙 시/기록할 지) : 戠은 소리와 창으로 이루어져 진흙에 새겨 표시한다는 뜻입니다. 말(言)이나 기호로 표시하여 안다는 뜻입니다.
마음속의 뜻(意)을 스스로 깨달아 아는(識) 상태를 의미합니다.

横断
횡단(おうだん)
横 (가로 횡) = 木 (나무 목) + 黄 (누를 황) : 누런(黄) 불빛이 넓게 퍼지듯 나무(木)가 가로로 넓게 놓인 모습에서 가로를 뜻합니다.
断 (끊을 단) = 斤 (도끼 근) + 㡭 (끊을 단의 변형) : 실타래를 도끼(斤)로 단호하게 내리쳐 끊는다는 뜻입니다.
가로(横) 방향으로 질러서 끊고(断) 지나가는 것을 의미합니다.

格好
격식, 꼴(かっこう)
格 (격식 격) = 木 (나무 목) + 各 (각각 각) : 나무(木)가 각각(各)의 모양에 맞게 일정한 틀이나 구조를 형성하는 규격이나 격식을 뜻합니다.
好 (좋을 호) = 女 (여자 녀) + 子 (아들 자) : 여자(女)가 아이(子)를 안고 있는 모습에서 서로 좋아하고 아름답다는 뜻이 되었습니다.
일정한 틀이나 격식(格)에 맞추어 겉으로 보기에 좋은(好) 모양이나 꼴을 의미합니다.

完成
완성(かんせい)
完 (완전할 완) = 宀 (집 엄) + 元 (으뜸 원) : 집(宀)을 짓는 일을 으뜸(元)가는 상태로 끝마쳤다는 데서 완전하다는 뜻이 되었습니다.
成 (이룰 성) = 戊 (창 무) + 丁 (장정 정) : 장정(丁)이 무기(戊)를 들고 평정하여 일을 이루어냈다는 뜻입니다.
어떤 일을 부족함 없이 완전하게(完) 이루어(成) 끝내는 것을 뜻합니다.

計画
계획(けいかく)
計 (셀 계) = 言 (말씀 언) + 十 (열 십) : 열(十) 번을 거듭하여 말(言)로 꼼꼼히 셈하고 따진다는 뜻입니다.
画 (그림 화/그을 획) = 聿 (붓 율) + 田 (밭 전) + 一 (한 일) : 붓(聿)으로 밭(田)의 경계를 긋는 모습에서 그림이나 구획을 짜는 것을 의미합니다.
앞으로 할 일을 미리 셈하여(計) 밑그림을 그리듯 틀(画)을 짜는 것을 의미합니다.

効果
효과(こうか)
効 (본받을 효/힘쓸 효) = 交 (사귈 교) + 力 (힘 력) : 힘(力)을 들이고 사귀어(交) 좋은 보람이나 결과가 나타난다는 뜻입니다.
果 (열매 과) = 木 (나무 목) + 田 (밭 전의 변형) : 나무(木) 위에 둥근 열매가 맺힌 모습에서 결과나 과실을 뜻합니다.
힘을 들인 결과로 나타나는 좋은 보람(効)과 맺혀진 결실(果)을 의미합니다.

終点
종점(しゅうてん)
終 (끝날 종) = 糸 (실 사) + 冬 (겨울 동) : 실(糸)을 다 짜고 겨울(冬)이 되어 한 해가 끝났다는 데서 마무리하다는 뜻입니다.
点 (점 점) = 黑 (검을 흑) + 占 (점칠 점) : 검은(黑) 색으로 점(占)을 찍어 표시한다는 뜻입니다.
어떤 길이나 과정이 모두 끝나는(終) 마지막 지점(点)을 의미합니다.

情報
정보(じょうほう)
情 (뜻 정) = 忄 (마음 심) + 青 (푸를 청) : 마음(忄)속 깊은 곳에서 우러나오는 푸른(青) 생명력처럼 순수하고 참된 감정이나 뜻을 의미합니다.
報 (알릴 보) = 幸 (다행 행) + 卩 (병부 절) + 𠬤 (또 우) : 다행스러운(幸) 소식을 사람(卩)에게 손(𠬤)으로 전하여 알린다는 뜻입니다.
어떤 사정이나 뜻(情)을 다른 사람에게 널리 알리는(報) 소식을 뜻합니다.

推薦
추천(すいせん)
推 (밀 추) = 扌 (손 수) + 隹 (새 추) : 손(扌)으로 새(隹)를 밀어 날려 보내듯 앞으로 밀고 나간다는 뜻입니다.
薦 (천거할 천) = 艹 (풀 초) + 廌 (해태 치) : 짐승(廌)이 먹는 풀(艹)을 모아 바친다는 데서 좋은 것을 천거하다는 뜻이 되었습니다.
어떤 사람이나 사물을 앞으로 밀어내어(推) 남에게 좋다고 바치며 권하는(薦) 것을 의미합니다.

相互
상호(そうご)
相 (서로 상) = 木 (나무 목) + 目 (눈 목) : 나무(木)의 생장 상태를 눈(目)으로 자세히 살펴본다는 뜻에서 바탕이나 서로를 뜻하게 되었습니다.
互 (서로 호) = 끈이나 실을 얼기설기 교차시켜 감아놓은 얼레의 모습에서 서로 얽혀있음을 의미합니다.
서로(相) 얼기설기 얽혀서(互) 관계를 맺고 있는 양쪽 모두를 뜻합니다.

遊学
유학(ゆうがく)
遊 (놀 유) = 辶 (책받침) + 斿 (깃발 유) : 깃발(斿)을 들고 길을 따라(辶) 여기저기 돌아다니며 논다는 뜻입니다.
学 (배울 학) = 𦥯 (배울 학의 원형) + 子 (아들 자) : 아이(子)가 양손으로 덮개를 덮고 가르침을 받는 모습에서 배우다는 뜻입니다.
자기 고향을 떠나 다른 곳으로 돌아다니며(遊) 학문이나 기술을 배우는(学) 것을 뜻합니다.

知識
지식(ちしき)
知 (알 지) = 矢 (화살 시) + 口 (입 구) : 화살(矢)처럼 빠르게 입(口)으로 말할 수 있을 만큼 사물을 명확히 꿰뚫어 안다는 뜻입니다.
識 (알 식) = 言 (말씀 언) + 戠 (찰흙 시/기록할 지) : 찰흙에 새기듯 말(言)이나 기호로 표시하여 사물을 분별하고 안다는 뜻입니다.
사물의 이치를 깨달아 알고(知) 분별하여 내 것으로 만드는(識) 것을 뜻합니다.

道路
도로(どうろ)
道 (길 도) = 辶 (책받침) + 首 (머리 수) : 사람의 머리(首)가 향하는 방향을 따라 길을 간다(辶)는 뜻에서 길을 의미합니다.
路 (길 로) = 足 (발 족) + 各 (각각 각) : 여러 사람의 발(足)이 각각(各) 디디고 지나가서 생긴 길을 뜻합니다.
사람이나 차가 다닐 수 있도록 만들어진 길(道)과 통로(路)를 뜻합니다.

圧力
압력(あつりょく)
圧 (누를 압) = 厂 (기슭 엄) + 土 (흙 토) : 산기슭(厂)의 흙(土)이 무너져 내려 무언가를 짓누른다는 의미에서 파생된 약자입니다.
力 (힘 력) = 쟁기나 사람의 팽팽한 근육을 본뜬 상형자로 외부로 발산되는 강한 힘을 뜻합니다.
위에서 아래로 내리누르는(圧) 강한 힘(力)을 의미합니다.

異常
이상(いじょう)
異 (다를 이) = 畀 (줄 비) + 廾 (두 손 공)의 변형으로, 귀신이나 괴물이 가면을 쓴 모습에서 본래와 다르고 기괴하다는 뜻이 되었습니다.
常 (항상 상) = 尙 (오히려 상) + 巾 (수건 건) : 수건처럼 항상 몸에 지니고 다닌다는 데서 평상시를 뜻합니다.
항상 그러한 평상시(常)의 상태와 다르고(異) 정상적이지 않음을 뜻합니다.

影響
영향(えいきょう)
影 (그림자 영) = 景 (볕 경) + 彡 (터럭 삼/무늬) : 햇볕(景)에 의해 생기는 무늬(彡)에서 그림자를 뜻합니다.
響 (울릴 향) = 郷 (고향 향) + 音 (소리 음) : 고향(郷) 마을 사람들이 잔치를 할 때 나는 소리(音)에서 널리 울려 퍼진다는 뜻입니다.
그림자(影)가 드리우고 소리가 울려(響) 퍼지듯, 어떤 사물이나 행동이 다른 것에 미치는 작용을 의미합니다.

会見
회견(かいけん)
会 (모일 회) = 人 (사람 인) + 云 (구름 운) : 사람들이 구름(云)처럼 한곳에 모인다(人)는 뜻입니다.
見 (볼 견) = 目 (눈 목) + 儿 (어진사람 인) : 사람(儿)이 눈(目)으로 무언가를 바라보고 만난다는 뜻입니다.
사람들이 공식적인 자리에서 함께 모여(会) 얼굴을 마주 보고 만나는(見) 것을 뜻합니다.

観光
관광(かんこう)
観 (볼 관) = 雚 (황새 관) + 見 (볼 견) : 황새(雚)처럼 눈을 크게 뜨고 자세히 두루 살펴본다(見)는 뜻입니다.
光 (빛 광) = 火 (불 화) + 儿 (어진사람 인) : 사람(儿)이 횃불(火)을 들고 있는 모습에서 빛이나 훌륭한 풍경을 뜻합니다.
다른 지역의 훌륭한 풍경이나 문화(光)를 널리 두루 살펴보는(観) 것을 의미합니다.

形式
형식(けいしき)
形 (모양 형) = 幵 (평평할 견) + 彡 (터럭 삼/무늬) : 평평한 틀에 무늬(彡)를 새긴 모습에서 겉모양을 뜻합니다.
式 (법 식) = 弋 (주살 익) + 工 (장인 공) : 장인(工)이 기준에 맞게 무언가를 만드는 법칙(式)을 뜻합니다.
사물이 겉으로 나타나는 모양(形)과 그것을 담고 있는 일정한 틀(式)을 의미합니다.

高級
고급(こうきゅう)
高 (높을 고) = 높게 지어진 누각의 상형자로 높거나 뛰어나다는 뜻입니다.
級 (등급 급) = 糸 (실 사) + 及 (미칠 급) : 실(糸)의 품질이 어디에 미치는지(及) 따져 매긴 등급을 의미합니다.
물건의 품질이나 지위 등이 매겨진 등급(級) 중에서 매우 높은(高) 상태를 뜻합니다.

社会
사회(しゃかい)
社 (모일 사) = 礻 (보일 시) + 土 (흙 토) : 흙(土) 제단(礻)에 마을 사람들이 모이는 모습에서 공동체를 뜻합니다.
会 (모일 회) = 人 (사람 인) + 云 (구름 운) : 사람들이 구름처럼 모인다는 뜻입니다.
공동의 목적을 위해 사람들이 한곳에 모여(会) 이룬 거대한 집단이나 공동체(社)를 뜻합니다.

将来
장래(しょうらい)
将 (장차 장) = 爿 (조각 널 장/평상) + 月 (고기 육) + 寸 (마디 촌/손) : 제단(爿)에 고기(月)를 올리며 기원하는 모습에서 장차 다가올 일을 뜻합니다.
来 (올 래) = 보리가 하늘에서 내려왔다는 상형자에서 오다라는 의미가 되었습니다.
장차(将) 다가올 미래의 시간이나 앞으로 다가올(来) 앞날을 의미합니다.

制限
제한(せいげん)
制 (절제할 제) = 未 (아닐 미/나뭇가지) + 刂 (칼 도) : 칼(刂)로 나뭇가지를 쳐내어 모양을 통제한다는 뜻입니다.
限 (한계 한) = 阝 (언덕 부) + 艮 (그칠 간) : 험한 언덕(阝) 앞에서 발걸음을 멈춘다(艮)는 한계를 뜻합니다.
범위를 넘지 못하도록 통제하여(制) 일정한 한계(限)를 두는 것을 뜻합니다.

速度
속도(そくど)
速 (빠를 속) = 束 (묶을 속) + 辶 (책받침) : 짐을 단단히 묶어(束) 빠르게 길을 나아간다(辶)는 뜻입니다.
度 (법도 도) = 广 (집 엄) + 廿 (스물 입) + 又 (손) : 손으로 길이나 양을 재는 법도나 비율을 뜻합니다.
사물이 이동하거나 진행되는 빠른(速) 비율(度)을 의미합니다.

探索
탐색(たんさく)
探 (찾을 탐) = 扌 (손 수) + 罙 (깊을 심) : 손(扌)을 깊은(罙) 곳까지 뻗어 찾는다 뜻입니다.
索 (찾을 색) = 十 (열 십) + 冖 (덮을 멱) + 糸 (실 사) : 덮여 있는 실 뭉치를 풀어 필요한 것을 찾는 모습입니다.
숨겨진 사물을 깊이 더듬고(探) 뒤져서 찾아내는(索) 것을 의미합니다.

超過
초과(ちょうか)
超 (뛰어넘을 초) = 走 (달릴 주) + 召 (부를 소) : 달리는(走) 사람을 부를(召) 정도로 훌쩍 뛰어넘는다는 뜻입니다.
過 (지날 과) = 辶 (책받침) + 咼 (비뚤어질 과) : 길을 가다가(辶) 바른길을 벗어나 지나쳐버렸다는 뜻입니다.
일정한 기준이나 한계를 훌쩍 뛰어넘어(超) 지나쳐버린(過) 상태를 뜻합니다.

特色
특색(とくしょく)
特 (특별할 특) = 牛 (소 우) + 寺 (절 사) : 절(寺)에서 제물로 쓰기 위해 고른 소(牛)에서 특별하다는 뜻이 되었습니다.
色 (빛 색) = 사람이 다른 사람의 등에 업힌 모습에서 겉으로 드러나는 빛깔을 뜻합니다.
보통의 것과 다르게 특별히(特) 눈에 띄는 겉모습이나 고유한 성질(色)을 뜻합니다.

案
안(あん)
案 (책상 안/생각 안) = 安 (편안할 안) + 木 (나무 목) : 나무(木)로 만들어져 물건을 편안히(安) 올려두는 책상이나 그 위에서 짜내는 생각을 의미합니다.
일을 처리하기 위해 내놓은 구체적인 생각이나 계획을 뜻하는 단어입니다.

移動
이동(いどう)
移 (옮길 이) = 禾 (벼 화) + 多 (많을 다) : 벼(禾) 모종을 많은(多) 곳에서 빈 곳으로 옮겨 심는다는 뜻입니다.
動 (움직일 동) = 重 (무거울 중) + 力 (힘 력) : 무거운(重) 짐을 힘(力)을 주어 옮기며 움직인다는 뜻입니다.
원래 있던 자리에서 다른 곳으로 위치를 옮기고(移) 움직이는(動) 것을 의미합니다.

援助
원조(えんじょ)
援 (도울 원) = 扌 (손 수) + 爰 (당길 원) : 구덩이에 빠진 사람을 손(扌)으로 당겨(爰) 돕는다는 뜻입니다.
助 (도울 조) = 且 (제기) + 力 (힘 력) : 제기를 들고 제사를 지낼 때 힘(力)을 보태어 거든다는 뜻입니다.
어려움에 처한 사람에게 손을 당겨주며(援) 힘을 보태어 돕는(助) 행위를 의미합니다.

過程
과정(かてい)
過 (지날 과) = 辶 (책받침) + 咼 (비뚤어질 과) : 길(辶)을 따라가며 어떤 지점을 지나쳐 간다는 뜻입니다.
程 (한도 정/길 정) = 禾 (벼 화) + 呈 (드릴 정) : 벼(禾)의 수확량을 재어 드리는(呈) 일정한 규정이나 길을 뜻합니다.
어떤 일이 진행되어 나가는 길(程)이나 거쳐서 지나가는(過) 단계를 의미합니다.

勘定
감정, 계산(かんじょう)
勘 (헤아릴 감) = 甚 (심할 심) + 力 (힘 력) : 힘(力)을 심하게(甚) 들이며 사실을 깊이 헤아린다는 뜻입니다.
定 (정할 정) = 宀 (집 엄) + 正 (바를 정) : 집 안에서 물건을 바른 위치에 고정시켜 정한다는 뜻입니다.
수량이나 금액을 깊이 헤아려(勘) 셈하여 정하는(定) 것을 의미합니다.

機械
기계(きかい)
機 (틀 기) = 木 (나무 목) + 幾 (기미 기) : 나무(木)로 만들어진 복잡하고 미세한(幾) 장치를 뜻합니다.
械 (형틀 계/기계 계) = 木 (나무 목) + 戒 (경계할 계) : 경계하기(戒) 위해 나무(木)로 만든 형틀이나 도구를 뜻합니다.
동력을 사용하여 복잡한 작업을 수행하도록 만들어진 도구나 틀을 뜻합니다.

構造
구조(こうぞう)
構 (얽을 구) = 木 (나무 목) + 冓 (짤 구) : 나무(木)를 엇갈리게 얽어서(冓) 물건을 뼈대 있게 짠다는 뜻입니다.
造 (지을 조) = 辶 (책받침) + 告 (알릴 고) : 길을 가며 사람들에게 널리 알리듯 규모를 크게 지어 만든다는 뜻입니다.
부분들을 서로 얽어서(構) 전체적인 형태나 뼈대로 만들어 낸(造) 짜임새를 뜻합니다.

失業
실업(しつぎょう)
失 (잃을 실) = 手 (손 수) + 乙 (새 을) : 손(手)에 쥐고 있던 새(乙)를 놓쳐 잃어버렸다는 뜻입니다.
業 (업 업) = 丵 (풀무성할 착) + 木 (나무 목) : 나무 받침대 위에서 정성 들여 하는 일이나 생업을 뜻합니다.
종사하던 직업이나 생업(業)을 잃어버린(失) 상태를 의미합니다.

図
그림(ず)
図 (그림 도) = 囗 (에운담 국) + 㐅 (교차할 오) : 경계(囗) 안에 엑스 자(㐅)를 그려 지형이나 구역을 나타낸다는 뜻입니다.
사물의 형태나 위치를 점이나 선으로 그려 나타낸 것을 의미하는 단어입니다.

総額
총액(そうがく)
総 (거느릴 총) = 糸 (실 사) + 忩 (바쁠 총) : 흩어진 실들을 하나로 모두 묶어 거느린다는 뜻으로 전부를 뜻합니다.
額 (액수 액) = 客 (손 객) + 頁 (머리 혈) : 사람의 두드러진 이마를 뜻하며 나중에 금전의 수량을 뜻하게 되었습니다.
흩어져 있는 금액을 모두 묶어(総) 합친 금전의 전체 액수(額)를 뜻합니다.

存在
존재(そんざい)
存 (있을 존) = 子 (아들 자) + 才 (바탕 재) : 아이(子)가 세상에 태어나 바탕(才)을 가지고 살아 있다는 뜻입니다.
在 (있을 재) = 土 (흙 토) + 才 (바탕 재) : 흙(土)으로 만든 바탕 위에 자리 잡고 있다는 뜻입니다.
사물이나 사람이 세상에 실체를 가지고(存) 자리 잡아 있는(在) 상태를 뜻합니다.

担任
담임(たんにん)
担 (멜 담) = 扌 (손 수) + 旦 (아침 단) : 손(扌)으로 무거운 짐을 어깨에 둘러메거나 떠맡는다는 뜻입니다.
任 (맡길 임) = 亻 (사람 인) + 壬 (짊어질 임) : 사람(亻)에게 짐을 짊어지게(壬) 하여 책임을 맡긴다는 뜻입니다.
특정한 학급이나 업무에 대한 책임을 짊어지고(担) 오롯이 맡아(任) 관리하는 사람을 뜻합니다.

道具
도구(どうぐ)
道 (길 도) = 辶 (책받침) + 首 (머리 수) : 머리(首)가 향하는 방향을 따라 길을 간다(辶)는 뜻입니다.
具 (갖출 구) = 貝 (조개 패) + 廾 (두 손 공) : 제기를 두 손(廾)으로 받들어 정성껏 갖춰 놓은 모습입니다.
목적이나 길(道)을 이루기 위해 수단으로 갖추어(具) 사용하는 기구나 연장을 뜻합니다.

燃焼
연소(ねんしょう)
燃 (탈 연) = 火 (불 화) + 然 (그러할 연) : 불(火)이 기세 좋게 타오른다는 뜻입니다.
焼 (사를 소) = 火 (불 화) + 尭 (높을 요) : 불(火)이 흙 가마(尭)를 달구어 뜨겁게 태운다는 뜻입니다.
물질이 불빛이나 열을 내며 불길에 타오르는(燃) 현상을 의미합니다.

能力
능력(のうりょく)
能 (능할 능) = 곰의 모습을 본뜬 상형자로 곰의 억센 힘에서 무언가를 할 수 있다는 뜻이 되었습니다.
力 (힘 력) = 근육이나 쟁기를 본뜬 글자로 발산되는 힘을 뜻합니다.
어떤 일을 감당하고 해낼 수 있는(能) 육체적 혹은 정신적인 힘(力)을 의미합니다.

破壊
파괴(はかい)
破 (깨뜨릴 파) = 石 (돌 석) + 皮 (가죽 피) : 돌(石)로 가죽(皮)을 찢고 산산조각 낸다는 뜻입니다.
壊 (무너질 괴) = 土 (흙 토) + 褱 (품을 회) : 흙(土)으로 만든 형태가 허물어진다는 뜻입니다.
형체를 갖춘 사물이나 체제를 깨뜨리고(破) 허물어뜨려(壊) 못 쓰게 만드는 것을 뜻합니다.

物価
물가(ぶっか)
物 (만물 물) = 牛 (소 우) + 勿 (말 물) : 얼룩소(牛)의 털 색이 섞여 구별할 수 없다는 데서 다양한 사물을 뜻합니다.
価 (값 가) = 亻 (사람 인) + 賈 (장사할 고) : 사람(亻)이 장사하며 물건에 가치나 가격을 매긴다는 뜻입니다.
거래되는 물건(物)에 매겨진 가격(価)이나 그 평균 수준을 의미합니다.

本来
본래(ほんらい)
本 (근본 본) = 木 (나무 목) + 一 (한 일) : 나무(木)의 뿌리 부분에 선(一)을 그어 사물의 바탕을 뜻합니다.
来 (올 래) = 보리의 모양을 본뜬 글자로 하늘에서 다가온다는 뜻입니다.
사물이 처음 생겨날 때부터(来) 가지고 있는 바탕이나 근본(本) 그대로를 의미합니다.

免許
면허(めんきょ)
免 (면할 면) = 투구를 쓴 사람의 모습에서 위험을 피하고 벗어나 벗었다는 의미입니다.
許 (허락할 허) = 言 (말씀 언) + 午 (낮 오) : 낮(午)에 밝은 곳에서 공명정대하게 말(言)로 허락한다는 뜻입니다.
금지당했던 상태를 면제하여(免) 벗어나게 해주고 자유롭게 하도록 허락하는(許) 것을 의미합니다.

有利
유리(ゆうり)
有 (있을 유) = 𠂇 (왼손 좌) + 月 (고기 육) : 손(𠂇)에 제물용 고기(月)를 쥐고 있다는 데서 가지고 있다는 뜻입니다.
利 (이로울 리) = 禾 (벼 화) + 刂 (칼 도) : 다 익은 벼(禾)를 낫(刂)으로 베어 이득을 얻는다는 뜻입니다.
자신에게 이득(利)이 되는 조건이나 상태를 가지고 있는(有) 것을 의미합니다.

理解
이해(りかい)
理 (다스릴 리) = 玉 (구슬 옥) + 里 (마을 리) : 옥(玉)을 결에 따라 잘 다듬고 이치를 따진다는 뜻입니다.
解 (풀 해) = 角 (뿔 각) + 刀 (칼 도) + 牛 (소 우) : 소(牛)의 뿔(角)을 칼(刀)로 쪼개어 얽힌 것을 푼다는 뜻입니다.
사물의 이치(理)를 논리적으로 쪼개고 풀어내어(解) 온전히 깨닫는 것을 의미합니다.

礼儀
예의(れいぎ)
礼 (예절 례) = 礻 (보일 시) + 乚 (새 을) : 신에게 제사(礻)를 지낼 때 갖추는 몸가짐에서 예절을 뜻합니다.
儀 (거동 의) = 亻 (사람 인) + 義 (옳을 의) : 사람(亻)이 옳고 바르게(義) 행동하는 겉모습이나 본보기를 뜻합니다.
사회 생활을 하며 지켜야 할 바른 몸가짐(礼)과 본보기적 행동(儀)을 의미합니다.

流行
유행(りゅうこう)
流 (흐를 류) = 氵 (물 수) + 㐬 (깃발 류) : 물(氵)이 깃발(㐬)이 나부끼듯 막힘없이 퍼져 흐른다는 뜻입니다.
行 (다닐 행) = 사거리를 뻗은 길의 모양을 본뜬 글자로 길을 다닌다는 뜻입니다.
어떤 풍속이 물결처럼 널리 흘러(流) 세상에 퍼져 돌아다니는(行) 것을 뜻합니다.

息抜き
휴식(いきぬき)
息 (숨 쉴 식) = 自 (코) + 心 (마음 심) : 코(自)와 심장(心)이 함께 작용하여 숨을 쉬는 생명 활동을 뜻합니다.
抜 (뺄 발) = 扌 (손 수) + 友 (벗 우) : 손(扌)으로 막힌 것을 뽑아내어 빠져나가게 한다는 뜻입니다.
긴장하여 꽉 차 있던 숨(息)을 부드럽게 빼내어(抜) 긴장을 풀고 쉬는 것을 뜻합니다.

型
형(かた)
型 (거푸집 형) = 刑 (형벌 형/모양) + 土 (흙 토) : 흙(土)으로 정해진 모양(刑)을 빚어 만든 틀을 뜻합니다.
어떤 물건을 일정한 모양으로 찍어내는 틀이나 일정한 형식을 뜻하는 단어입니다.

職場
직장(しょくば)
職 (직분 직) = 耳 (귀 이) + 戈 (창 과) + 𠂇 (손) : 창(戈)을 쥐고 임무를 수행하는 직분을 뜻합니다.
場 (마당 장) = 土 (흙 토) + 昜 (볕 양) : 볕(昜)이 드는 넓은 흙(土) 마당으로 장소를 뜻합니다.
고유한 임무(職)를 수행하기 위해 출근하는 물리적 현장(場)을 뜻합니다.

縁
연(えん)
縁 (가선 연) = 糸 (실 사) + 彖 (단 단) : 옷 가장자리를 실(糸)로 두른 모습에서 맺어진 인연을 의미하게 되었습니다.
사람과 사람 사이에 보이지 않는 끈으로 이어지고 맺어진 관계를 뜻하는 단어입니다.

合図
신호(あいず)
合 (합할 합) = 亼 (모일 집) + 口 (입 구) : 뚜껑(亼)이 그릇(口) 입구에 꼭 맞게 닫혀 합쳐진다는 뜻입니다.
図 (그림 도) = 囗 (에운담 국) + 㐅 (교차할 오) : 구역(囗) 안에 기호(㐅)를 그려 뜻을 나타내는 모양입니다.
서로의 뜻이 맞게 합쳐지도록(合) 미리 정해둔 기호나 그림(図) 같은 몸짓을 의미합니다.

勢い
기세, 기운(いきおい)
勢 (형세 세) = 埶 (심을 예) + 力 (힘 력) : 나무를 땅에 튼튼하게 심어놓은(埶) 모습처럼 뻗어 나가는 힘(力)을 뜻합니다.
사물이나 현상이 세차게 뻗어 나가는 물리적, 심리적인 힘이나 기운을 뜻하는 단어입니다.

表
표(ひょう)
表 (겉 표) = 毛 (털 모) + 衣 (옷 의) : 짐승의 털로 짠 옷의 겉면이라는 뜻에서 밖으로 드러나는 표를 의미합니다.
복잡한 내용을 겉으로 드러나게 보기 좋게 정리해 놓은 양식을 뜻하는 단어입니다.

小箱
소박스(こばこ)
小 (작을 소) = 세 개의 점으로 무언가가 작게 갈라진 모양을 본뜬 상형자입니다.
箱 (상자 상) = 竹 (대죽) + 相 (서로 상) : 대나무(竹)를 엮어서 맞물리게 짠 바구니나 상자를 의미합니다.
물건을 담기 위해 만든 상자(箱) 중 크기가 작은(小) 것을 의미합니다.

値段
가격(ねだん)
値 (값 치) = 亻 (사람 인) + 直 (곧을 직) : 사람(亻)이 물건의 곧고(直) 합당한 가치를 매긴 값이라는 뜻입니다.
段 (구빌 단) = 殳 (몽둥이 수) + 叚 (빌릴 가) : 산비탈을 몽둥이로 다져 계단처럼 층계를 만든다는 뜻입니다.
물건의 가치에 따라 매겨진(値) 화폐의 액수나 등급(段)을 의미합니다.

血液型
혈액형(けつえきがた)
血 (피 혈) = 皿 (그릇 명) + 丶 (불똥 주) : 제사용 그릇(皿)에 희생양의 피(丶)가 떨어지는 모습입니다.

申す (말씀드리다) = 윗사람에게 뜻을 아룀.
込む (넣다) = 안으로 깊숙이 들어감.
자신의 의사나 요구사항을 서류나 말로 정리하여 기관이나 상대방의 안쪽으로 정중하게 제출하고 청하는 행위를 뜻하는 복합 동사입니다.

持ち越す
넘기다, 미루다(もちこす)
持つ (들다/가지다) = 손에 쥠.
越す (넘다) = 시간이나 경계를 넘어감.
어떤 일이나 문제를 당장 해결하지 않고 손에 쥔 채로 다음 기회로 넘겨서 미루어 두는 행위를 뜻하는 복합 동사입니다.

接する
접하다(せっする)
接 (이을 접) = 扌 (손 수) + 妾 (가까이 할 첩) : 손(扌)을 뻗어 곁에 두는 사람(妾)처럼 가까이 맞닿고 잇는다는 뜻입니다.
두 사물이나 사람이 손을 맞대듯 가까이 맞닿아 이어지는 것을 의미하는 동사입니다.

優しい
다정하다, 상냥하다(やさしい)
優 (뛰어날 우) = 亻 (사람 인) + 憂 (근심 우) : 憂는 머리(頁)와 마음(心)과 천천히 걷는 발(夂)로 이루어져 근심하며 걷는 모습입니다. 근심하는(憂) 사람(亻)을 위로할 만큼 넉넉하고 뛰어나며 상냥하다는 뜻입니다.
남을 배려하고 마음씨가 따뜻하며 상냥한 성질을 뜻하는 형용사입니다.

確認する
확인하다(かくにんする)
確 (굳을 확) = 石 (돌 석) + 隺 (두루미 학) : 두루미(隺)가 돌(石)처럼 단단한 곳에 확실히 자리 잡는다는 뜻입니다.
認 (알 인) = 言 (말씀 언) + 忍 (참을 인) : 참고 꼼꼼히 살핀 뒤 말(言)로 인정한다는 의미입니다.
어떤 사실이나 상태가 틀림이 없는지 단단하게 짚어보고 맞다고 인정하는 것을 의미하는 동사입니다.

制限
제한(せいげん)
制 (절제할 제) = 未 (아닐 미/나뭇가지) + 刂 (칼 도) : 칼(刂)로 나뭇가지를 쳐내어 모양을 통제한다는 뜻입니다.
限 (한계 한) = 阝 (언덕 부) + 艮 (그칠 간) : 험한 언덕(阝) 앞에서 발걸음을 멈춘다(艮)는 한계를 뜻합니다.
범위를 넘지 못하도록 통제하여 일정한 한계를 두는 것을 뜻합니다.

制度
제도(せいど)
制 (절제할 제) = 나뭇가지를 칼로 쳐내어 규격에 맞게 통제하고 억제한다는 뜻입니다.
度 (법도 도) = 广 (집 엄) + 廿 (스물 입) + 又 (손) : 손으로 길이나 양을 재는 법도나 규칙을 뜻합니다.
국가나 사회의 목적을 달성하기 위해 규격에 맞게 통제하고 정해 놓은 법도나 규칙을 의미합니다.

辺り
근처, 부근(あたり)
辺 (가 곁 변) = 辶 (책받침) + 刀 (칼 도) : 칼(刀)로 물건을 벨 때 생기는 가장자리나 길(辶)의 가장자리를 뜻하는 신자체입니다.
어떤 특정한 장소나 대상의 가장자리, 즉 그 주변이나 근처를 뜻하는 명사입니다.

勢い
기세, 기운(いきおい)
勢 (형세 세) = 埶 (심을 예) + 力 (힘 력) : 나무를 땅에 튼튼하게 심어놓은(埶) 모습처럼 밖으로 뻗어 나가는 굳센 힘(力)을 뜻합니다.
사물이나 현상이 세차게 뻗어 나가는 물리적, 심리적인 힘이나 기운을 뜻하는 명사입니다.

丘
언덕(おか)
丘 (언덕 구) = 땅 위에 불룩하게 솟아오른 둥근 산이나 언덕의 모양을 본뜬 상형자입니다.
평지보다 조금 높게 솟아오른 작은 산이나 땅을 뜻하는 단일 한자어입니다.

読もて
읽다(よむ) 또는 독자(よみて)의 오타 추정
読 (읽을 독) = 言 (말씀 언) + 売 (팔 매) : 말(言)을 하나하나 이어서 풀이한다는 뜻입니다. (売는 賣의 약자로 이어진다는 의미로 쓰임)
글자나 문장을 소리 내어 말하거나 뜻을 헤아려 이해하는 행위를 뜻합니다. 제시해주신 단어는 読む(よむ) 혹은 読み手(よみて)의 오타로 보여 해당 한자를 기준으로 설명했습니다.

型
형, 틀(かた)
型 (거푸집 형) = 刑 (형벌 형/모양) + 土 (흙 토) : 흙(土)으로 정해진 모양(刑)을 빚어 만든 틀을 뜻합니다.
어떤 물건을 일정한 모양으로 찍어내는 틀이나 일정한 형식을 뜻하는 명사입니다.

焦点
초점(しょうてん)
焦 (탈 초) = 隹 (새 추) + 灬 (불 화) : 새(隹)가 불(灬) 위에서 그을려 탄다는 뜻입니다.
点 (점 점) = 黑 (검을 흑) + 占 (점칠 점) : 검은 색으로 점을 찍어 표시한다는 뜻입니다.
불에 타서 그을린 점처럼 빛이 모이는 중심점이나, 사람들의 관심이 모이는 가장 핵심적인 부분을 뜻합니다.

改める
고치다, 변경하다(あらためる)
改 (고칠 개) = 己 (몸 기) + 攴 (칠 복) : 자신의 몸이나 상태(己)를 회초리로 치듯(攴) 다스려 바르게 잡는다는 뜻입니다.
잘못된 것이나 낡은 것을 쳐내어 새롭게 고치고 바로잡는 것을 뜻하는 동사입니다.
改善

개선(かいぜん)
改 (고칠 개) = 낡은 상태를 쳐서 바르게 고친다는 뜻입니다.
善 (착할 선) = 羊 (양 양) + 言 (말씀 언) 두 개가 합쳐진 형태에서 유래하여 양처럼 온순하고 훌륭한 말을 한다는 데서 좋다는 뜻이 되었습니다.
잘못된 것이나 부족한 점을 고치고 바로잡아(改) 더 좋은(善) 상태로 만드는 것을 의미합니다.

致
하다의 겸양어, 이르다(いたす)
致 (이를 치) = 至 (이를 지) + 攴 (칠 복) : 목적지에 다다르기(至) 위해 채찍질하여(攴) 나아가게 한다는 뜻입니다.
어떤 일이나 결과를 이끌어내다, 혹은 하다(する)의 동작을 공손하게 낮추어 표현하는 동사 致す(いたす)로 주로 쓰입니다.

飾る
장식하다(かざる)
飾 (꾸밀 식) = 食 (밥 식) + 巾 (수건 건) : 밥(食)을 먹을 때 상을 수건(巾)으로 닦고 깔끔하게 꾸민다는 뜻입니다.
밋밋한 물건이나 장소를 보기 좋게 꾸미고 화려하게 장식하는 행위를 뜻하는 동사입니다.

崩れる
무너지다(くずれる)
崩 (무너질 붕) = 山 (뫼 산) + 朋 (벗 붕) : 산(山)처럼 거대한 것이 무거워서 와르르 무너져 내린다는 뜻입니다.
차곡차곡 쌓여 있거나 형태를 갖추고 있던 것이 형체를 잃고 허물어지는 것을 뜻하는 동사입니다.

暮れる
저물다(くれる)
暮 (저물 모) = 莫 (없을 막) + 日 (날 일) : 해(日)가 풀밭 아래로 넓게 숨어 날이 어두워진다는 뜻입니다.
태양이 지고 날이 어두워지거나 한 해나 계절이 끝나가는 것을 뜻하는 동사입니다.

祭り
축제, 제사(まつり)
祭 (제사 제) = 月 (고기 육) + 又 (손) + 示 (제단) : 손(又)으로 고기(月)를 제단(示) 위에 올려놓고 기원하는 모습입니다.
신에게 제물을 바치며 기원하는 의식이나, 오늘날 많은 사람이 모여 즐기는 축제를 뜻하는 명사입니다.

際
때, 즈음, 곁(さい / きわ)
際 (가 즈음 제) = 阝 (언덕 부) + 祭 (제사 제) : 언덕(阝)에서 제사(祭)를 지내며 신과 인간의 경계가 맞닿는 시점이나 그 사이를 뜻합니다.
어떤 일이 일어나는 바로 그 시점이나 기회, 혹은 사물과 사물이 맞닿는 경계를 뜻하는 명사입니다.

失う
잃다(うしなう)
失 (잃을 실) = 手 (손 수) + 乙 (새 을) : 손(手)에 쥐고 있던 새(乙)를 놓쳐서 날려 보내 잃어버렸다는 뜻입니다.
본래 가지고 있던 물건이나 기회, 사람 등을 놓치거나 빼앗겨서 더 이상 가지지 못하게 되는 것을 뜻하는 동사입니다.
